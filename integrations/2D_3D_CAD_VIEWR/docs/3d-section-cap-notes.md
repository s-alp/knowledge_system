# 3D 断面キャップメモ

## 結論

断面キャップは `閉じた STL メッシュ` を対象にします。直接 STL と `STEP -> STL` の両方で適用できますが、開いたメッシュでは保証しません。

## 実装方針

- 描画は Three.js の `stencil clipping` ベースで行う
- 本体メッシュは clip plane で切る
- 切断面には cap plane を重ねて表示する
- 単一平面断面のみ対象
- 軽量化はキャップ以外の要素で吸収し、断面 ON 時の中実表示は維持する

## キャップ対象

- 閉じた STL
- STEP 由来で STL に変換された閉メッシュ

## キャップ対象外

- 境界 edge が 2 回出現しない開いたメッシュ
- 壊れた STL
- STEP 変換失敗で `failed` になったジョブ

## UI 挙動

- 閉メッシュなら断面 ON 時にキャップを描画する
- 開いたメッシュなら「断面キャップ未対応」と表示する
- 輪郭強調は切替式で、既定は OFF
- 重いケースでは輪郭強調より断面キャップを優先する
