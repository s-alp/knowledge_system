SxSys.setShade メソッド (Boolean)

アクティブウィンドウのCG表示状態を変更します。(CG表示/WIRE表示)
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Shared Sub setShade( _
   ByVal shade As Boolean _
)
[C#]
public static void setShade(
   bool shade
);

パラメータ

shade
表示方法。
true : CG表示
false : WIRE表示

解説

アクティブウィンドウの表示状態(CG表示/WIRE表示)の変更を行います。
アクティブウィンドウが3次元の場合に使用して下さい。
shade=trueの場合、現在のシステムのCG表示モードに従って、CG表示されます。

指定した要素のみを、CG表示/WIRE表示に切り替えたい場合は、 SxEntクラスのsetShadeを使用して下さい。
システムのCG表示モードのみを変更したい場合は、setShadeModeで可能です。
