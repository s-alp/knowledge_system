検索状態
true : 検索
false : 非検索

解説

自インスタンスの全フィールドを指定属性に設定します。
CAD上の検索色の状態は変更しません。
CAD上の検索色の設定を行うには、SxSysクラスの、 setSearchColorメソッドで行います。
