SxModel.setDim メソッド 

自モデルをアクティブモデルに設定し、表示を2次元又は3次元に切り替えます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Sub setDim( _
   ByVal is3d As Boolean _
)
[C#]
public void setDim(
   bool is3d
);

パラメータ

is3d
次元。
true:3次元。
false:2次元。

解説

アクティブウィンドウが自モデルを表示している場合、アクティブウィンドウの表示を切り替えます。
アクティブウィンドウが別モデルを表示している場合、自モデルを表示する基本ウィンドウを アクティブウィンドウに設定し、表示を切り替えます。
