SxWindow メソッド

SxWindow クラスのメソッドは以下のとおりです。SxWindow クラスの完全なメンバ一覧については、 SxWindow メンバ のトピックを参照してください。

静的 パブリック (共有) メソッド

getActive アクティブウィンドウを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getMDIRect 図形表示領域を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getWindowList 開設中のウィンドウを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

パブリック メソッド

closeオーバーロード。 ウィンドウを閉じます。(C++用)
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
create 図形表示ウィンドウの開設を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
deleteオーバーロード。 ウィンドウを閉じます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getActiveAsmPlane アクティブ組立平面を取得します。
getAsmPlaneList 組立平面のリストを取得します。
getCoordVirtual スクリーン座標をウィンドウに表示されているVS/WFの空間座標に変換します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getInf ウィンドウの情報を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getMatrix ウィンドウのローテーョンマトリクス(回転行列)を取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getStatusAsmPlaneオーバーロード。 指定組立平面の表示状態を取得します。
isEqual 指定SxWindowインスタンスが自ウィンドウとCAD上の同一ウィンドウを指しているか判定する。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
move ウィンドウを移動します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
pan 画面のパニングを行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
resize ウィンドウのリサイズを行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
rotateオーバーロード。 画面の回転（ローテーション）を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setActive アクティブウィンドウに設定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setActiveAsmPlane 指定組立平面をアクティブに設定します。
setDim ウィンドウの次元を切り替えます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setInvisibleExcAsmPlane 指定組立平面以外の組立平面を非表示にします。
setStatus ウィンドウの表示状態を変更します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setSysView 3次元システムビュー表示を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setUserView 3次元ユーザービュー表示を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setVisibleAsmPlane 指定組立平面の表示状態を設定します。
setVS ウィンドウに指定VSを表示し、アクティブウィンドウに設定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setWF ウィンドウに指定WFを表示し、アクティブウィンドウに設定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
zoomオーバーロード。 拡大/縮小表示を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
zoomful 全画面表示を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
