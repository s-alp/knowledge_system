SxEntPart.getParent メソッド 

親パーツを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function getParent() As SxEntPart
[C#]
public SxEntPart getParent();

戻り値

親パーツ
親パーツが存在しない場合、null(VBの場合、Nothing)を返答します。

解説

自パーツの親パーツを問い合わせます。
親パーツが存在しない場合、null(VBの場合、Nothing)を返します。
第一階層のパーツ(最上位パーツ直下のパーツ)の場合、親パーツは無しです。
自パーツの子パーツのリストを問い合わせる場合は、getChildList メソッドを使用してください。

Exception その他の例外
