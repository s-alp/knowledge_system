パラメータ

mode
作成モード
true:配置子図を作成します。
false:子図複写を行います。
falseの場合、子図の複写を実行し、null(VBの場合Nothing)を返答します。
vs
参照先のVS
pos
配置座標
Z値は無視されます。

戻り値

作成したレファー(配置子図)

解説

アクティブウィンドウのアクティブVSに配置子図を作成します。
アクティブウィンドウには、引数vsの属するモデルの2次元が表示されている必要があります。
vsのタイプは子図(SxVS.TYPE_WORK_VS)である必要があります。
(ビュー、ローカルビュー、グローバルビューは指定できません)
mode=falseの場合、子図複写を実行し、null(VBの場合、Nothing)を返答します。

写像部品の作成は、SxFilePartクラスの、open2D メソッドを参照して下さい。
