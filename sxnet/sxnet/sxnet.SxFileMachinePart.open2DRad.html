パラメータ

mode
処理モード
SxFileMachinePart.MODE_SEG : 実要素化
SxFileMachinePart.MODE_RPART : 実像部品化
SxFileMachinePart.MODE_GRP : グループ化
pattern
呼び名
pos
配置座標
ang
回転角度(ラジアン)
layer
レイヤ引継ぎ(true:引き継ぐ, false:入力レイヤに作成)
reverse_x
X軸反転(true:反転します, false:反転しません)
reverse_y
Y軸反転(true:反転します, false:反転しません)

戻り値

配置した要素
mode=SxFileMachinePart.MODE_SEGの場合、nullを返答します。CAD上で部品配置自体は行われます。
mode=SxFileMachinePart.MODE_PARTの場合、SxEntRPartインスタンスを返答します。
mode=SxFileMachinePart.MODE_GRPの場合、SxEntGrpインスタンスを返答します。

解説

アクティブウィンドウのアクティブVSに、2次元機械部品(2次元 JIS部品)の配置を行います。
