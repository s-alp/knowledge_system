パブリック コンストラクタ

SxDraftAtrText コンストラクタ インスタンスの初期化

パブリック フィールド

color 文字の色
font フォント種別。
true:TrueTypeFont文字
false:ストローク文字
font_name フォント名。
font=trueの場合のみ有効です。
height 文字の高さ
row_space 行間隔
space_ratio 文字の間隔比率
tilt 文字の斜体角度(度)（-45 〜 45)
TrueTypeFontの場合、0又は-15
width 文字の線幅
width_ratio 文字の幅比率
