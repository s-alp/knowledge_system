SxEnt.setVisible メソッド (SxEnt[], Boolean)

指定要素群の表示属性を一括変更します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Shared Sub setVisible( _
   ByVal entlist As SxEnt(), _
   ByVal visible As Boolean _
)
[C#]
public static void setVisible(
   SxEnt[] entlist,
   bool visible
);

パラメータ

entlist
表示属性を変更する要素の配列
entlist = NULLの場合、全要素を処理対象とします。
visible
表示属性
true:表示
false:非表示

解説

指定要素群の表示状態を一括変更します。
半透明表示中の要素は処理対象外になります。
entlistには、アクティブVS/WF内の要素を指定して下さい。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
