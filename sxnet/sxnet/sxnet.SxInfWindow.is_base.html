SxInfWindow.is_base フィールド

ウィンドウ種別。
true:基本ウィンドウ
false:一時ウィンドウ

[Visual Basic]
Public is_base As Boolean
[C#]
public bool is_base;

解説

基本ウィンドウは、モデルを新規作成、又はロードした時に開かれるウィンドウです。
基本ウィンドウを閉じた場合、同一モデルを表示する全てのウィンドウと、 モデルを閉じます。
一時ウィンドウはを閉じた場合は、そのウィンドウのみを閉じます。
