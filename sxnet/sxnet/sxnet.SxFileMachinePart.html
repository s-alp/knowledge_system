スレッドセーフ

この型の public static (Visual Basicでは Shared) のすべてのメンバは、 マルチスレッド操作で安全に使用できます。インスタンスのメンバの場合は、スレッドセーフであるとは限りません。

解説

機械部品ファイル(JIS部品、3Dパラメトリック部品）にアクセスするためのクラスです。
機械部品ファイルの情報取得、CAD上への部品配置処理を行います。
3DJIS部品の場合、同一フォルダ内に、〜@3DPARAM.icdファイルが存在する場合は、 3DJIS部品ファイルである、〜@3DPARAM.icdを指定して下さい。
〜@3DPARAM.icdが存在しない場合は、2D部品と共通の部品ファイル〜@正面.icd又は〜@平面.icd等を指定して下さい。
2DJIS部品の場合は、〜@正面.icd又は〜@平面.icdを指定して下さい。

使用例

ユーザ定義部品Test@3DPARAM.icdを指定位置に配置する。[C#] 

 using System;
 using sxnet;

 class Test
 {
        static void Main(string[] args)
        {
            try
            {
                // アクティブモデルの表示を3次元に切り替える。
                SxModel act_model = SxModel.getActive();
                act_model.setDim(true);
                
                // 機械部品ファイルオブジェクトの生成
                SxFileMachinePart part = 
                    new SxFileMachinePart("C:\\ICADSXPRO\\JISPARTS\\USER\\TEST\\TEST@3DPARAM.icd");

                // 配置座標と、配置方向の設定
                SxPos pos = new SxPos(10.0e0, 0.0e0, 0.0e0);
                SxVec zvec = new SxVec( 0.0e0, 1.0e0, 0.0e0 );
                SxVec xvec = new SxVec( 1.0e0, 0.0e0, 0.0e0);

                // レイヤ引継ぎ(ON)
                bool layer = true;

                // 呼び名のリストを問い合わせる
                string[] pattern = part.getPatternNames();

                // 指定呼び名のパラメタ値リストを問い合わせる
                double[] values = part.getVarValues(pattern[0]);

                // 第一パラメタの値を100.0に変更する。
                values[0] = 100.0e0;
                part.setVarValues(pattern[0], values);

                // パーツ化して部品を配置する。
                SxEntPart ent = (SxEntPart)part.open( SxFileMachinePart.MODE_PART, 
                                            pattern[0], pos, zvec, xvec, layer );
            }
            catch( SxException e )
            {
                Console.WriteLine(e.Message);
            }
            catch( Exception e )
            {
                Console.WriteLine(e.Message);
            }
        }
    }

3DJIS部品(3D専用部品)の配置サンプル。[C#] 
 using System;
 using sxnet;

 class Test
 {
        static void Main(string[] args)
        {
            try
            {
                // アクティブモデルの表示を3次元に切り替える。
                SxModel act_model = SxModel.getActive();
                act_model.setDim(true);
                
                // 機械部品ファイルオブジェクトの生成
                //    四角ボルト・上は、3D専用部品ファイル"四角ボルト・上@3DPARAM.icd"を持つので、
                //    3D専用部品ファイルを指定する。
                SxFileMachinePart part = 
                    new SxFileMachinePart(SxFileMachinePart.getDir() + "\\3D\\02_ボルト\\四角ボルト\\四角ボルト・上",
                        "四角ボルト・上@3DPARAM.icd");

                // 配置座標と、配置方向の設定
                SxPos pos = new SxPos(10.0e0, 0.0e0, 0.0e0);
                SxVec zvec = new SxVec( 0.0e0, 1.0e0, 0.0e0 );
                SxVec xvec = new SxVec( 1.0e0, 0.0e0, 0.0e0);

                // レイヤ引継ぎ(ON)
                bool layer = true;

                // パーツ化して呼び名="M10"で部品を配置する。
                SxEntPart ent = (SxEntPart)part.open( SxFileMachinePart.MODE_PART, 
                                            "M10", pos, zvec, xvec, layer );
            }
            catch( SxException e )
            {
                Console.WriteLine(e.Message);
            }
            catch( Exception e )
            {
                Console.WriteLine(e.Message);
            }
        }
    }
