SxWF.getActive メソッド 

アクティブウィンドウに表示されている、アクティブWFを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getActive() As SxWF
[C#]
public static SxWF getActive();

戻り値

SxWFインスタンス。
アクティブウィンドウが2次元の場合、null(VBの場合、Nothing)を返します。

解説

CAD上での現在のアクティブWFを取得します。
ウィンドウが存在しない、又は、アクティブウィンドウが2次元を表示している場合、 null(VBの場合、Nothing)を返します。
