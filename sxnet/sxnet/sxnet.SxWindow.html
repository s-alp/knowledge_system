この型の public static (Visual Basicでは Shared) のすべてのメンバは、 マルチスレッド操作で安全に使用できます。インスタンスのメンバの場合は、スレッドセーフであるとは限りません。

解説

CAD上の図形表示ウィンドウにアクセスするためのクラス。
ウィンドウの開設や、移動、リサイズ、画面操作処理等、ウィンドウに関する各種操作を行います。

CAD上で開設済みのウィンドウリストの取得は、 本クラスのgetWindowListメソッドを参照して下さい。
指定モデルを表示するウィンドウの取得は、SxModelクラスの、 getWindowメソッドを参照して下さい。
指定VSを表示するウィンドウの取得は、SxVSクラスの、 getWindowメソッドを参照して下さい。
指定WFを表示するウィンドウの取得は、SxWFクラスの、 getWindowメソッドを参照して下さい。
指定要素を表示するウィンドウの取得は、SxEntクラスの、 getWindowメソッドを参照して下さい。
注意事項：ウィンドウを閉じた場合、CAD上でウィンドウの序列が変更になる為、 既に生成済みの他のSxWindowインスタンスも無効になります。

例として、以下のような処理で、問題が発生します。
(前提条件として、CAD上で2枚のウィンドウが開かれているものとします。) 

[C#]

    // 開設済みのウィンドウを取得する。
    SxWindow[] list = SxWindow.getWindowList();
    
    // ウィンドウを閉じる。
    list[0].delete();
    
    // 別のウィンドウの情報を得る。
    // → 前の処理でウィンドウを閉じているので、list[]内のウィンドウインスタンスは
    //    CAD上のウィンドウと整合が取れていない為、正常に処理できません。
    SxInfWindow inf = list[list.Length-1].getInf();
このような場合、ウィンドウを閉じた後に、再度、ウィンドウインスタンスの取得が必要になります。
以下は修正例です。
[C#]

    // 開設済みのウィンドウを取得する。
    SxWindow[] list = SxWindow.getWindowList();
    
    // ウィンドウを閉じる。
    list[0].delete();

    // 開設済みのウィンドウを再取得する。
    list = SxWindow.getWindowList();
        
    // 別のウィンドウの情報を得る。
    SxInfWindow inf = list[list.Length-1].getInf();
