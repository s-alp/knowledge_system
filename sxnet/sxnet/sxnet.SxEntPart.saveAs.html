SxEntPart.saveAs メソッド 

自パーツ、および配下のパーツ（外部パーツを含む）を1つのファイルとして保存します。

[Visual Basic]
Public Sub saveAs( _
   ByVal path As String, _
   ByVal fname As String, _
   ByVal comment As String _
)
[C#]
public void saveAs(
   string path,
   string fname,
   string comment
);

パラメータ

path
保存フォルダパス。
空文字列""を指定した場合、元バーツが存在したフォルダに保存します。
fname
ファイル名(拡張子なし)。
空文字列""を指定した場合、ファイル名=バーツ名として保存します。
ファイル名の最大長は半角英数字の場合40文字までです。(全角文字で20文字)
comment
コメント文字列。
空文字列""が指定された場合、コメント無しで保存します。
コメント文字列の最大長は半角英数字の場合80文字までです。(全角文字で40文字)

解説

自パーツ配下に外部パーツが含まれる場合、外部パーツは取り込まれ（内部パーツ化され）、1つのファイルになります。
pathとfnameの長さの合計が、半角英数字で255文字以内である必要があります。
自パーツは、アクティブウィンドウに表示されている必要があります。
