SxSys.init メソッド (Int32)

ローカルマシン上のCADとの通信初期化を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Shared Sub init( _
   ByVal portno As Integer _
)
[C#]
public static void init(
   int portno
);

パラメータ

portno
CADと接続するポート番号

解説

ポート番号には、.NET連携セットアップで定義した、 sxnetのポート番号を指定して下さい。
本メソッドは、プログラムの先頭で必ず１回呼び出して下さい。
以下の状態で本メソッドを実行した場合、例外が発生します。
・CADが起動中でない場合。
・ローカルマシン上の%WINDIR%\SYSTEM32\DRIVERS\ETC\SERVICES内に、 sxnetのポート番号が定義されていない。あるいは、ポート番号が本メソッドで指定した ポート番号と異なる場合。
・ファイアウォール等によって、指定したポート番号の通信がブロックされている場合。

接続が確立できない場合、メッセージボックスにエラーの内容を表示します。

例外

例外の種類 条件
Exception CADとの接続に失敗しました。
