SX.Net Class Library

SxVS メソッド

SxVS クラスのメソッドは以下のとおりです。SxVS クラスの完全なメンバ一覧については、 SxVS メンバ のトピックを参照してください。

静的 パブリック (共有) メソッド

getActive アクティブウィンドウに表示されている、アクティブVSを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getViewName 基本ビュータイプから対応するビュー名を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setCoordオーバーロード。 指定VS群の３次元配置情報を設定します。

パブリック メソッド

close VSを削除します。(C++用)
convExtent 自VS上のエクステントを、グローバルビュー上のエクステントに変換します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
convPos 座標変換(ビュー/ローカルビュー ←→ グローバルビュー)を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
delete VSを削除します。
deleteDraftLine アクティブビュー、またはアクティブ子図内の下書き線を削除します。
getEntList 要素の矩形検索を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getExtent 自VSのエクステントを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getGrp 自VS内の指定名のグループを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getGrpList 自VS内のグループを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getInf VS情報を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getModel 親モデルを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getReferList 自VS内のレファー(写像部品/配置子図)を取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getRPartListオーバーロード。 自VS内の指定名の実像部品を取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getSegListオーバーロード。 自VS内のセグメントを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getWindow 自VSを表示するウィンドウを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
isEqual 指定SxVSインスタンスが自インスタンスとCAD上の同一VSを指しているか問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
isValid 自VSがCAD上で有効か問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
move ビュー/ローカルビューの配置位置を変更します。
rotate ローカルビューの配置角度を設定します。
setActiveオーバーロード。 アクティブVSに設定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setComment 子図にコメントを設定します。
setCoordオーバーロード。 VSへ３次元配置情報を設定します。
setName 子図、ローカルビューの子図名、ビュー名を変更します。
setOrigin ビュー/ローカルビューの原点を変更します。
子図の基準点を変更します。
setScale ビュー/ローカルビュー/子図の尺度を設定します。
