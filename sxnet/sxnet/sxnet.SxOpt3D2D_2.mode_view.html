SxOpt3D2D_2.mode_view フィールド

基本ビュー配置モード
SxOpt3D2D_2.MODE_VIEW_AUTO : 自動配置
SxOpt3D2D_2.MODE_VIEW_NOCHANGE : 既存のビュー位置固定(デフォルト)
SxOpt3D2D_2.MODE_VIEW_MANUAL : ビュー配置位置指定

SxWF.project3D2D_2使用時のみ有効です。

[Visual Basic]
Public mode_view As Integer
[C#]
public int mode_view;

解説

3D→2D/展開図作成時の基本ビューの配置位置指定モード。
SxWF.project3D2D_2使用時のみ有効です。
SxOpt3D2D_2.MODE_VIEW_AUTOの場合、焼き付ける要素のエクテントと 指定された余白値から、ビューの配置位置を自動計算します。
SxOpt3D2D_2.MODE_VIEW_NOCHANGEの場合、開設済のビューの配置位置は 変わりません。未開設のビューは配置位置を自動計算します。
SxOpt3D2D_2.MODE_VIEW_MANUALの場合、指定された位置にビューを配置します。
