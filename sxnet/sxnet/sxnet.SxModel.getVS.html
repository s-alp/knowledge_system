SxModel.getVS メソッド 

モデル内の指定名のVSを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function getVS( _
   ByVal name As String _
) As SxVS
[C#]
public SxVS getVS(
   string name
);

パラメータ

name
VS名(半角英数字で最大8文字まで)

戻り値

SxVSインスタンス。
存在しない場合、null(VBの場合、Nothing)を返します。

解説

自モデル内の指定名のVS(子図、ビュー)を取得します。
指定名のVSが存在しない場合、null(VBの場合、Nothing)を返答します。

例外
