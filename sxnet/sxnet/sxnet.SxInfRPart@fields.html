SxInfRPart フィールド

SxInfRPart クラスのフィールドは以下のとおりです。SxInfRPart クラスの完全なメンバ一覧については、 SxInfRPart メンバ のトピックを参照してください。

パブリック フィールド

angle 回転角度(度)。
参照図面が設定されている場合のみ有効です。
comment コメント。
コメントなしの場合、空文字列""です。
is_mirror ミラー属性。
true:ミラーされています。
false:ミラーされていません。
基準座標を持つ場合のみ有効です。
name 実像部品名
part3d_name 図面化時の元の3Dパーツ名。
3D→2Dコマンドでパーツを図面化して作成された実像部品の場合の元パーツ名です。
パーツの図面化で作成されていない場合は、空文字列("")です。
pos 基準座標。
Z座標は常に0.0です。
基準座標が設定されていない場合、null(VBの場合、Nothing)です。
ref_model_name 参照図面名。
未設定の場合、空文字列""です。
ref_vs_name 参照VS名。
未設定の場合、空文字列""です。
