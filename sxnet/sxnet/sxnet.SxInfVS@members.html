SxInfVS 概要

パブリック コンストラクタ

SxInfVS コンストラクタ 指定SxInfインスタンスのコピーを作成します。

パブリック フィールド

angle グローバルビュー上の配置角度(度)。 (ローカルビューの場合のみ有効です)
子図の場合、基準点とレバー点の角度(配置角度)です。
comment コメント文字列。 子図の場合のみ有効です。
未設定の場合は、空文字列""です。
cs 3次元配置情報(グローバル座標系)。
(VSタイプはローカルビューの場合のみ有効です)
3次元配置情報なしの場合はnull(VBの場合Nothing)を返します。
name VS名
org グローバルビュー上の配置原点。(対象VSがビュー、ローカルビューの場合のみ有効です)
子図の場合、基準点です。
2次元座標値です。(Z値は常に0.0です)
scale 尺度
type VSタイプ。
SxVS.TYPE_WORK_VS : 子図
SxVS.TYPE_GLOBAL_VIEW : グローバルビュー
SxVS.TYPE_VIEW : 基本ビュー
SxVS.TYPE_LOCAL_VIEW : ローカルビュー
TYPE_WORK_VS TYPE_GLOBAL_VIEW TYPE_VIEW TYPE_LOCAL_VIEW
view_type 基本ビュータイプ。
VSのタイプが基本ビューの場合のみ有効です。
基本ビュータイプ定数は、SxVS.VIEW_xxx定数を参照して下さい。
