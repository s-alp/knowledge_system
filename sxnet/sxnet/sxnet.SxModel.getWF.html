SxModel.getWF メソッド 

モデル内の指定WF名のWFを取得する。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function getWF( _
   ByVal name As String _
) As SxWF
[C#]
public SxWF getWF(
   string name
);

パラメータ

name
WF名(半角英数字で最大8文字まで)

戻り値

SxWFインスタンス。
存在しない場合は、null(VBの場合、Nothing)を返します。

解説

自モデル内の指定名を持つWFを問い合わせます。
指定名のWFが存在しない場合、null(VBの場合、Nothing)を返答します。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
