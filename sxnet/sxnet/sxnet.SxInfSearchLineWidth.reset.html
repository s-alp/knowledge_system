パラメータ

onoff
検索属性
true : 検索
false : 非検索

解説

自インスタンスの全フィールドを指定値に設定します。
CAD上の検索線幅の状態は変化しません。
CAD上の検索線幅の状態を変更するには、SxSysクラスの、 setSearchLineWidthメソッドを使用します。
