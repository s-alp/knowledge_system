静的 パブリック (共有) フィールド

STREAK_N 筋目方向記号定数：Ｎ
STREAK_NONE 筋目方向記号定数：無し

proway 加工方法
remove_type 除去加工タイプ

SxGeomSmark.REMOVTYP_REMOVE : 除去要
SxGeomSmark.REMOVTYP_NODEL : 除去禁止
streak_dir 筋目方向記号
