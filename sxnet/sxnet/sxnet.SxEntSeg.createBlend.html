3次元立体形状の丸み/面取りを作成します。

Public Sub createBlend( _
   ByVal edge_list As SxEdge(), _

[C#]
public void createBlend(
   SxEdge[] edge_list,
   double radius,
   int mode
);

パラメータ

edge_list
丸み/面取りを行う自要素のエッジの配列。
null(VBの場合、Nothing)の場合、自要素の全エッジを対象とすします。
radius
半径／面取り長さ
mode
処理種別:
SxEntSeg.BLEND_BALL : 丸み
SxEntSeg.BLEND_CHAMFER : 面取り

解説

自要素の指定エッジに対して、丸み/面取り編集を行います。
自要素が、立体、ソリッド、FCソリッドの場合、処理対象です。
edge_list[]には、自要素のエッジを指定して下さい。
非表示要素は処理対象外です。
