この型の public static (Visual Basicでは Shared) のすべてのメンバは、 マルチスレッド操作で安全に使用できます。インスタンスのメンバの場合は、スレッドセーフであるとは限りません。

解説

セグメント、フェイス、エッジの各ジオメトリ情報クラスの抽象クラスです。
インスタンスは、本クラスから派生したクラスで生成します。

SxEntSegオブジェクトのgetGeomメソッドにより、 各セグメントタイプ別のジオメトリ情報を取得できます。
SxFaceオブジェクトのgetGeomメソッドにより、 フェイスタイプ別のジオメオトリ情報を取得できます。
SxEdgeオブジェクトのgetGeomメソッドにより、 エッジタイプ別のジオメトリ情報を取得できます。
