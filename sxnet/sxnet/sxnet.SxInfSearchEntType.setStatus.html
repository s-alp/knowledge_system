SxInfSearchEntType.setStatus メソッド 

自インスタンス内の指定要素タイプの検索属性の設定を行います。

[Visual Basic]
Public Sub setStatus( _
   ByVal enttyp As Integer, _

)
[C#]
public void setStatus(
   int enttyp,
   bool onoff
);

パラメータ

enttyp
要素タイプ番号
onoff
検索属性
true:検索
false:非検索

解説

自インスタンス内の指定要素タイプ番号に応じたフィールドの検索属性の設定を行います。
CAD上の要素検索属性の状態は変わりません。
enttypに対象外の番号を指定した場合、何もしません。
要素タイプ番号については、SxEntSegクラスの、SEGTYPE_xxx定数を参照して下さい。
CAD上の検索要素タイプの設定を行うには、SxSysクラスの、 setSearchEntTypeメソッドを使用します。
