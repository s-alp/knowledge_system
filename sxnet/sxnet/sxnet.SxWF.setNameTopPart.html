SxWF.setNameTopPart メソッド 

3DグローバルWFの最上位パーツ名、コメントの設定を行います。

[Visual Basic]
Public Sub setNameTopPart( _
   ByVal name As String, _
   ByVal comment As String _
)
[C#]
public void setNameTopPart(
   string name,
   string comment
);

パラメータ

name
パーツ名。
半角英数字で最大40文字まで。(全角文字で20文字まで)
comment
コメント。
半角英数字で最大48文字まで。(全角文字で24文字まで)
省略時は空文字列""を指定して下さい。

解説

3DグローバルWF内の最上位パーツの名前、コメントの設定を行います。
自WFがグローバルWFの場合のみ使用可能です。
自WFを表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
自WFを表示するウィンドウがアクティブウィンドウに設定されます。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
