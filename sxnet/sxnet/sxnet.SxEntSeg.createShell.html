パラメータ

face_list
開放面に指定するフェイスの配列
null(VBの場合、Nothing)の場合、開放面なしです。
shell_thick
共通の厚さ
mode
シェル化の方法
SxEntSeg.SHELL_INNER : 立体の面の内側に指定した厚さのシェルを作成します。
SxEntSeg.SHELL_OUTER : 立体の面の外側に指定した厚さのシェルを作成します。
SxEntSeg.SHELL_BOTH : 立体の面を中立面として両側に指定した厚さのシェルを作成します。

解説

自要素のシェル化を行います。
自要素が、立体、ソリッド、FCソリッドの場合、処理可能です。
face_list[]には、自要素のフェイスを指定して下さい。
非表示要素は処理対象外です。
