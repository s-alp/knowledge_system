SxSys.getCommand メソッド 

現在、CAD上で実行中のコマンドキーワードを取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getCommand() As String
[C#]
public static string getCommand();

戻り値

実行中のコマンドキーワード文字列。

解説

現在、CAD上で実行しているコマンドのコマンドキーワードを問い合わせます。
コマンドが実行中でない場合は、空文字列("")を返します。
尚、本メソッド実行により、CAD上でのコマンド実行は中止されます。
(何もコマンドが実行されていない状態に遷移します） iCAD SDK+での処理終了後に、処理開始前のコマンドに戻したい場合等に使用します。
使用例
[C#] 

    // 現在実行中のコマンドキーワードを取得する。
    string str = SxSys.getCommand();

    // 実行中のコマンド名をコマンドプロンプトに表示する。
    Console.WriteLine("コマンド名:" + str);

    // コマンド実行中の場合、元のコマンドに状態を戻す。
    if (str != "")
    {
        try
        {
            SxSys.command(str, false);
        }
        catch
        {
            ;
        }
    }
[VB] 
    ' 現在実行中のコマンドキーワードを取得する。
    Dim str As String = SxSys.getCommand()

    ' 実行中のコマンド名をコマンドプロンプトに表示する。
    Console.WriteLine("コマンド名:" + str)

    ' コマンド実行中の場合、元のコマンドに状態を戻す。
    If str <> "" Then
        Try
            SxSys.command(str, false);
        Catch

        End Catch
    End If
