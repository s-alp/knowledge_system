SxEntSeg.createRadiusDimStd3D メソッド 

半径寸法線を作成します。(3次元)

[Visual Basic]
Public Shared Function createRadiusDimStd3D( _
   ByVal pos As SxPos, _
   ByVal arcinf As Double(), _
   ByVal plane As SxGeomPlane _
) As SxEntSeg
[C#]
public static SxEntSeg createRadiusDimStd3D(
   SxPos pos,
   double[] arcinf,
   SxGeomPlane plane
);

パラメータ

pos
寸法配置点座標
arcinf
寸法線の基準円／円弧情報
arcinf[0] :基準円/円弧の中心点のＸ座標
arcinf[1] :基準円/円弧の中心点のＹ座標
arcinf[2] :基準円/円弧の中心点のＺ座標
arcinf[3] :基準円/円弧の半径
arcinf[4] :基準円/円弧の始角（単位：ラジアン。円の場合0.0を指定して下さい）
arcinf[5] :基準円/円弧の相対角（単位：ラジアン。円の場合は２πを指定して下さい）
plane
寸法線作成平面情報

戻り値

作成した半径寸法線セグメント

解説

アクティブウィンドウに表示されたWF上に、3次元半径寸法線を作成します。
planeで指定した平面上に寸法線が作成されます。
posは、3次元の座標です。これらをplaneで定義された平面上に投影して寸法線を作成します。
