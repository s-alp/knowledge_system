SxSys.getPosList メソッド 

CAD上での座標入力(複数)を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getPosList() As SxPos()
[C#]
public static SxPos[] getPosList();

戻り値

CAD上で指定された座標の配列。
処理がキャンセルされた場合、null(VBの場合、Nothing)を返答します。

解説

CAD上での座標入力操作(複数回)を行い、入力された座標の配列を返します。
CAD上で座標入力処理がキャンセルされた場合、null(VBの場合、Nothing)を返答します。
座標入力を行ったウィンドウに、アクティブな組立平面が存在し、かつ操作次元が2次元 の場合、座標は組立平面上の2次元座標系での値を返答します。
