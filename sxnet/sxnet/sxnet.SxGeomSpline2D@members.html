SxGeomSpline2D メンバ

SxGeomSpline2D 概要

パブリック コンストラクタ

SxGeomSpline2D コンストラクタ インスタンスを初期化します。

パブリック フィールド

eang 終点角度(度)
固定スプラインの場合のみ有効です。
kind スプライン種別
SxEntSeg.SPLINE_KIND_FIXED :固定スプライン
SxEntSeg.SPLINE_KIND_NORMAL :自然スプライン
SxEntSeg.SPLINE_KIND_PERIODIC :周期スプライン
pos 始点座標
Z値は0.0固定です。
sang 始点角度(度)
固定スプラインの場合のみ有効です。
type (SxGeom から継承されます) ジオメトリタイプ
vec_list 補間点座標（始点からの相対ベクトル）の配列
Z値は0.0固定です。
