この型の public static (Visual Basicでは Shared) のすべてのメンバは、 マルチスレッド操作で安全に使用できます。インスタンスのメンバの場合は、スレッドセーフであるとは限りません。

解説

SxEntPart.setInfExメソッド用のパラメタクラスです。 infoは、半角英数字で最大4000文字までです。(全角文字で最大2000文字まで)
自パーツが既に任意情報を持つ場合、上書きされます。

文字列は、以下の形式で指定して下さい。
項目ID1,"内容1",項目ID2,"内容2", ...
例）User_type,"A",User_weight,"200.0"

SxEntPart.setInfExメソッドで付加した情報は、CAD本体の３次元ツール/部品表コマンドで参照されます。
CAD本体の部品表コマンドで利用する情報を付加する場合は、定義したテンプレート の内容に従って、文字列をCSV形式で指定して下さい。
部品表コマンドの詳細については、CAD本体のヘルプを参照して下さい。
