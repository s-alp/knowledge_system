SxModel.save メソッド (String, String, String, Int32, Int32)

モデルをファイルに保存します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

パラメータ

path
保存フォルダパス。
空文字列""を指定した場合、元々ファイルが存在したフォルダに保存します。
fname
ファイル名(拡張子なし)。
空文字列""を指定した場合、ファイル名=モデル名として保存します。
ファイル名の最大長は半角英数字の場合40文字までです。(全角文字で20文字)
comment
コメント文字列。
空文字列""が指定された場合、コメント無しで保存します。
コメント文字列の最大長は半角英数字の場合80文字までです。(全角文字で40文字)
version
保存バージョン番号。
0を指定した場合、最新のバージョン/レベル形式で保存します。
level
保存レベル番号。
version=0の場合は、無視されます。

解説

モデルの保存を行います。
モデル内の読み取り専用の外部パーツが編集された場合、読み取り専用外部パーツは保存されません。
pathとfnameの長さの合計が、半角英数字で255文字以内である必要があります。
CAD上で開設した新規図面の場合、path,fnameは省略不可です。
version,levelは、例えばV5L4形式で保存する場合は、version=5, level=4を指定します。
最新V/Lから過去4世代のV/Lまでが指定可能です。
最新のV/Lで保存する場合は、version=0を指定して下さい。
