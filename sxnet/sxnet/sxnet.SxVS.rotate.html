SxVS.rotate メソッド 

ローカルビューの配置角度を設定します。

[Visual Basic]
Public Sub rotate( _
   ByVal angle As Double, _
   ByVal mode As Boolean _
)
[C#]
public void rotate(
   double angle,
   bool mode
);

パラメータ

angle
回転角度(度)
mode
処理モード。
true:座標軸とローカルビュー内の要素も合わせて回転します。 false:座標軸のみを回転します。

解説

自ローカルビューの配置角度を変更します。
自VSを表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
自VSを表示するウィンドウがアクティブウィンドウに設定されます。
自VSがアクティブビューに設定されます。
