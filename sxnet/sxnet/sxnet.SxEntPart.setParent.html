SxEntPart.setParent メソッド 

自パーツの親パーツを変更します。

[Visual Basic]
Public Sub setParent( _
   ByVal parent As SxEntPart _
)
[C#]
public void setParent(
   SxEntPart parent
);

パラメータ

parent
親パーツ
null(VBの場合Nothing)を指定した場合、最上位パーツ直下に階層変更を行います。

解説

パーツの階層を指定した親パーツ直下に変更します。
親パーツのアクセス権が読み取り専用であっても、親パーツの変更を実行します。
自パーツが入力パーツの場合、入力パーツは変更されます。
自パーツを表示するウィンドウをアクティブウィンドウに設定します。
自パーツを表示するウィンドウがない場合、新規にウィンドウを開設します。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
