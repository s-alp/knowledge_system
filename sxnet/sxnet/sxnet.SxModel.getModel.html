SxModel.getModel メソッド 

指定パス、指定モデル名を持つ開設済みモデルを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getModel( _
   ByVal path As String, _
   ByVal name As String _
) As SxModel
[C#]
public static SxModel getModel(
   string path,
   string name
);

パラメータ

path
モデルファイルの存在するフォルダパス(フルパス)
空文字列""の場合は、未保存のモデルとみなします。
name
モデル名(拡張子無し)

戻り値

モデルインスタンス。
指定モデルが開設されていない場合は、null(VBの場合は、Nothing)を返します。
