SxWF.setInfExTopPart メソッド 

3DグローバルWFの最上位パーツ任意情報の設定を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Sub setInfExTopPart( _
   ByVal info As String _
)
[C#]
public void setInfExTopPart(
   string info
);

パラメータ

info
付加情報文字列(CSV形式)

解説

3DグローバルWFの最上位パーツに任意情報を設定します。
infoは、半角英数字で最大4000文字までです。(全角文字で最大2000文字まで) 自WFは3DグローバルWFの場合のみ使用可能です。
最上位パーツが既に任意情報を持つ場合、上書きされます。

文字列は、以下の形式で指定して下さい。
項目ID1,"内容1",項目ID2,"内容2", ...
例）User_type,"A",User_weight,"200.0"

本メソッドで付加した情報は、CAD本体の３次元ツール/部品表コマンドで参照されます。
CAD本体の部品表コマンドで利用する情報を付加する場合は、定義済のテンプレート の内容に従って、文字列をCSV形式で指定して下さい。
部品表コマンドの詳細については、CAD本体のヘルプを参照して下さい。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外

使用例

[C#] 

void SetInf(SxWF wf)
{
    // 付加する任意情報文字列の定義
    string info = "User_Volume,\"100.0\",User_Weight,\"200.0\"";
    // WF内の最上位パーツに任意情報を付加する。
    wf.setInfEx(info);
}
[VB] 
Private Sub SetInf(ByVal wf As SxWF)
    ' 付加する任意情報文字列の定義
    Dim info As String = "User_Volume,""100.0"",User_Weight,""200.0"""
    ' WF内の最上位パーツに任意情報を付加する。
    wf.setInfEx(info)
End Sub
