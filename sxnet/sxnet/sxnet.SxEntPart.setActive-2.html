パラメータ

check_win
自パーツを表示するウィンドウが存在しない、又はアクティブウィンドウが2次元の場合の処理モード。
true : 自パーツを表示するウィンドウを開設し、アクティブウィンドウに設定します。
false : ウィンドウの開設を行いません。(アクティブウィンドウ上にある前提です)

解説

自パーツを入力パーツに設定します。
open_win = falseは、自パーツがアクティブウィンドウ上に表示されている場合のみ使用可能です。 trueの場合よりも、高速に処理を行う事ができます。
open_win = trueの場合は、自パーツを表示するウィンドウが存在しない、又は、アクティブウィンドウが 2次元の場合でも、自パーツを表示するウィンドウを開設し、アクティブウィンドウに設定します。setActive()メソッドと同じ動作です。
