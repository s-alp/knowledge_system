パラメータ

name
パーツ名
comment
コメント
コメントなしの場合、空文字列("")を指定して下さい。
filename
ファイル名(拡張子なし)
外部パーツの場合のみ有効です。
空文字列("")を指定した場合、パーツ名と同一ファイル名とします。
change_all
同名パーツの一括処理
true:同じ名前のパーツに対して、一括変更を行います。
false:自パーツのみ変更します。

解説

自パーツのパーツ名、コメントの設定を行います。
change_all=trueの場合、同一WF内の同じ名前を持つパーツに対して、一括変更を行います。
また、change_all = true、かつ、2次元に同一名の実像部品が存在する場合、実像部品名も一括変更します。 コメントは実像部品には反映されません。

パーツ名、ファイル名は、半角英数字で最大40文字までです。(全角文字の場合、最大20文字)
コメントは、半角英数字で最大48文字までです。(全角文字の場合、最大24文字)

自パーツを表示するウィンドウをアクティブウィンドウに設定します。
自パーツを表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
