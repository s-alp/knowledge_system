SxOptSurfmark フィールド

SxOptSurfmark クラスのフィールドは以下のとおりです。SxOptSurfmark クラスの完全なメンバ一覧については、 SxOptSurfmark メンバ のトピックを参照してください。

パブリック フィールド

additionalline 補助線付き
true: 補助線有
false: 補助線無（デフォルト）
lead_line = TRUE かつ lead_standard = TRUE 指定時のみ有効。
angle 角度(度)
setangle=trueの場合のみ有効。 (デフォルト値=60.0)
マイナス値は指定できません。
cinf 指示値、加工方法の文字列の配列
cinf[0]:指示値１
cinf[1]:指示値２
cinf[2]:指示値３
cinf[3]:指示値４
cinf[4]:加工方法
省略時は空文字列""を指定して下さい。(デフォルト)
jktype 除去加工タイプ
SxGeomSmark.REMOVTYP_NODEL : 除去禁止
SxGeomSmark.REMOVTYP_NOREMOVE : 除去不問（デフォルト)
SxGeomSmark.REMOVTYP_REMOVE : 除去要
lead_left 引出方向
true: 左
false: 右（デフォルト）
lead_line = TRUE 指定時のみ有効。
lead_line 引出線の有無
true: 引出線有
false: 引出線無（デフォルト）
lead_standard 引出モード
true: 標準（デフォルト）
false: 径／面取
lead_line = TRUE 指定時のみ有効。
setangle 角度指定
true: 角度指定あり
false: 角度指定無（デフォルト）
lead_line = TRUE かつ lead_standard = TRUE 指定時のみ有効。
shtype 筋目方向記号
SxGeomSmark.STREAK_NONE : 無し（デフォルト)
SxGeomSmark.STREAK_C : Ｃ
SxGeomSmark.STREAK_M : Ｍ
SxGeomSmark.STREAK_N : Ｎ
SxGeomSmark.STREAK_P : Ｐ
SxGeomSmark.STREAK_R : Ｒ
SxGeomSmark.STREAK_X : Ｘ
SxGeomSmark.STREAK_PAR : ＝
SxGeomSmark.STREAK_VER : ⊥
