SxParmAsmCoord メンバ

SxParmAsmCoord 概要

パブリック コンストラクタ

SxParmAsmCoord オーバーロード。 SxParmAsmCoord クラスの新しいインスタンスを初期化します。

パブリック フィールド

org 設定する基準座標。
null(VBの場合、Nothing)の場合、変更しません。
plane 処理対象の組立平面
xvec 設定するパーツ配置ベクトル(X軸単位ベクトル)。
null(VBの場合、Nothing)の場合、配置ベクトルは変更しません。
zvec 設定するパーツ配置ベクトル(Z軸単位ベクトル)。
null(VBの場合Nothing)の場合、基準座標のみ変更します。
