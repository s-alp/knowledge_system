SxOpt3D2D_2 メンバ

SxOpt3D2D_2 概要

静的 パブリック (共有) フィールド

CENTER_OFFSET_LENGTH JIS穴中心線作成のオフセット処理定数：長さ指定
CENTER_OFFSET_RATIO JIS穴中心線作成のオフセット処理定数：比率指定
HIDDEN_LINE_DRAFT 隠線処理定数：隠線を下書き線で作成する
HIDDEN_LINE_OFF 隠線処理定数：隠線を作成しない
HIDDEN_LINE_ON 隠線処理定数：隠線を作成する
HOLE_FIGURE 穴の図面化処理定数：形状焼付
HOLE_JIS 穴の図面化処理定数：JIS穴焼付
MODE_VIEW_AUTO 基本ビュー配置モード定数：自動
MODE_VIEW_MANUAL 基本ビュー配置モード定数：手動
MODE_VIEW_NOCHANGE 基本ビュー配置モード定数：固定
OUTLINE_DRAFT 外形線処理定数：下書き線化
OUTLINE_NORMAL 外形線処理定数：通常描画
PART_BOTTOM 実像部品化処理定数：末端パーツを実像部品化
PART_TOP 実像部品化処理定数：上位パーツを実像部品化
TANGENT_DRAFT 正接線処理定数：正接線を下書き線化
TANGENT_OFF 正接線処理定数：正接線除去
TANGENT_ON 正接線処理定数：正接線作図

パブリック コンストラクタ

SxOpt3D2D_2 コンストラクタ デフォルト設定でインスタンスを初期化します。

パブリック フィールド

center_offset JIS穴中心線作成のオフセット処理。
SxOpt3D2D_2.CENTER_OFFSET_RATIO : オフセット比率指定。(デフォルト)
SxOpt3D2D_2.CENTER_OFFSET_LENGTH : オフセット長さ指定。
copy3d 3D形状のコピー(将来の予約。現在未使用)
true:コピーします。(デフォルト)
false:コピーしません。
※現在、本設定は無視されます。
frame_file 図枠ファイル。
図枠配置なしの場合、nullを指定します。(デフォルト)
SxWF.project3D2D_2使用時のみ有効です。
2D図面が表示しているウィンドウが存在する、又は、2D図面のグローバルビューに要素が存在する場合、 本指定は無視されます。
hole JIS穴の図面化処理。
SxOpt3D2D_2.HOLE_FIGURE : 形状で穴を焼き付けます。(デフォルト)
SxOpt3D2D_2.HOLE_JIS : JIS表記で穴を焼き付けます。
hole_center_line JIS穴表記の中心線の有無。
true:中心線有り。
false:中心線無し。(デフォルト)
hole_mode = SxOpt3D2D_2.HOLE_JISの場合のみ有効です。
hole_inf JIS穴表記の場合の穴情報引継ぎ。
true:引継ぎ有り。
false:引継ぎ無し。(デフォルト)
hole_mode = SxOpt3D2D_2.HOLE_JISの場合のみ有効です。
inter_hidden_line 相互隠線処理。
SxOpt3D2D_2.HIDDEN_LINE_OFF : 隠線を作成しません。(デフォルト)
SxOpt3D2D_2.HIDDEN_LINE_ON : 隠線を作成します。
SxOpt3D2D_2.HIDDEN_LINE_DRAFT : 隠線を下書き線で作成します。
inter_hidden_line_style 相互隠線の線種。
inter_hidden_line = SxOpt3D2D_2.HIDDEN_LINE_OFFの場合、無効です。
SxSys.STYLE_SOLID : 実線
SxSys.STYLE_DASH : 破線 (デフォルト)
SxSys.STYLE_ONE_DOT_CHAIN : 一点鎖線
SxSys.STYLE_TWO_DOT_CHAIN : 二点鎖線
SxSys.STYLE_DOT : 点線
SxSys.STYLE_LONG_DASH : 長破線
inter_hidden_line_width 相互隠線の線幅。
inter_hidden_line = SxOpt3D2D_2.HIDDEN_LINE_OFFの場合、無効です。
0:3D形状の線幅を引き継ぎます。(デフォルト)
SxSys.WIDTH_BOLD : 太線
SxSys.WIDTH_MIDDLE : 中線
SxSys.WIDTH_THIN : 細線
mode_view 基本ビュー配置モード
SxOpt3D2D_2.MODE_VIEW_AUTO : 自動配置
SxOpt3D2D_2.MODE_VIEW_NOCHANGE : 既存のビュー位置固定(デフォルト)
SxOpt3D2D_2.MODE_VIEW_MANUAL : ビュー配置位置指定

SxWF.project3D2D_2使用時のみ有効です。
offset_length JIS穴中心線オフセット長さ。
初期値=2.0。
center_offset = SxOpt3D2D_2.CENTER_OFFSET_LENGTHの場合のみ有効です。
offset_ratio JIS穴中心線オフセット比率(%)
初期値=10.0。
center_offset = SxOpt3D2D_2.CENTER_OFFSET_RATIOの場合のみ有効です。
outline 外形線処理。
SxOpt3D2D_2.OUTLINE_NORMAL : 外形線を通常作図します。(デフォルト)
SxOpt3D2D_2.OUTLINE_DRAFT : 外形線を下書き線で作図します。
outline_style 外形線の線種。
0:3D形状の線種を引き継ぎます。(デフォルト)
SxSys.STYLE_SOLID : 実線
SxSys.STYLE_DASH : 破線
SxSys.STYLE_ONE_DOT_CHAIN : 一点鎖線
SxSys.STYLE_TWO_DOT_CHAIN : 二点鎖線
SxSys.STYLE_DOT : 点線
SxSys.STYLE_LONG_DASH : 長破線
outline_width 外形線の線幅。
0:3D形状の線幅を引き継ぎます。(デフォルト)
SxSys.WIDTH_BOLD : 太線
SxSys.WIDTH_MIDDLE : 中線
SxSys.WIDTH_THIN : 細線
part_mode 実像部品化する単位。
SxOpt3D2D_2.PART_BOTTOM : 末端パーツを実像部品化します。
SxOpt3D2D_2.PART_TOP : 上位パーツを実像部品化します。(デフォルト)
pos_view_back ビュー配置座標：背面図(-XYビュー)
SxWF.project3D2D_2使用時で、mode_view=SxOpt3D2D_2.MODE_VIEW_MANUAL、かつ、 view_back=trueの場合のみ有効です。
グローバルビュー上でのビュー配置座標を指定します。
pos_view_bottom ビュー配置座標：下面図(XZビュー)
SxWF.project3D2D_2使用時で、mode_view=SxOpt3D2D_2.MODE_VIEW_MANUAL、かつ、 view_bottom=trueの場合のみ有効です。
グローバルビュー上でのビュー配置座標を指定します。
pos_view_front ビュー配置座標：正面図(XYビュー)
SxWF.project3D2D_2使用時で、mode_view=SxOpt3D2D_2.MODE_VIEW_MANUAL、 かつ、view_front=trueの場合のみ有効です。
グローバルビュー上でのビュー配置座標を指定します。
pos_view_left ビュー配置座標：左側面図(ZYビュー)
SxWF.project3D2D_2使用時で、mode_view=SxOpt3D2D_2.MODE_VIEW_MANUAL、 かつ、view_left=trueの場合のみ有効です。
グローバルビュー上でのビュー配置座標を指定します。
pos_view_right ビュー配置座標：右側面図(-ZYビュー)
SxWF.project3D2D_2使用時で、mode_view=SxOpt3D2D_2.MODE_VIEW_MANUAL、かつ、 view_right=trueの場合のみ有効です。
グローバルビュー上でのビュー配置座標を指定します。
pos_view_top ビュー配置座標：平面図(X-Zビュー)
SxWF.project3D2D_2使用時で、mode_view=SxOpt3D2D_2.MODE_VIEW_MANUAL、 かつ、view_top=trueの場合のみ有効です。
グローバルビュー上でのビュー配置座標を指定します。
remove_int 干渉除去。
true:3D要素の干渉箇所を除去して焼き付けます。
false:3D要素の干渉を除去しません。(デフォルト)
remove_overlap 重複除去。
true:重複線の除去を行います。
false:重複線の除去を行いません。(デフォルト)
scale 尺度
0.0を指定した場合、図面に設定された尺度を引き継ぎます。
初期値=0.0。
SxWF.project3D2D_2使用時のみ有効です。
self_hidden_line 自己隠線処理。
SxOpt3D2D_2.HIDDEN_LINE_OFF : 隠線を作成しません。(デフォルト)
SxOpt3D2D_2.HIDDEN_LINE_ON : 隠線を作成します。
SxOpt3D2D_2.HIDDEN_LINE_DRAFT : 隠線を下書き線で作成します。
self_hidden_line_style 自己隠線の線種。
self_hidden_line = SxOpt3D2D_2.HIDDEN_LINE_OFFの場合、無効です。
SxSys.STYLE_SOLID : 実線
SxSys.STYLE_DASH : 破線 (デフォルト)
SxSys.STYLE_ONE_DOT_CHAIN : 一点鎖線
SxSys.STYLE_TWO_DOT_CHAIN : 二点鎖線
SxSys.STYLE_DOT : 点線
SxSys.STYLE_LONG_DASH : 長破線
self_hidden_line_width 自己隠線の線幅。
self_hidden_line = SxOpt3D2D_2.HIDDEN_LINE_OFFの場合、無効です。
0:3D形状の線幅を引き継ぎます。(デフォルト)
SxSys.WIDTH_BOLD : 太線
SxSys.WIDTH_MIDDLE : 中線
SxSys.WIDTH_THIN : 細線
space 余白指定の有無。
true:指定有り。
false:指定無し。自動計算。(デフォルト)
space_bottom 余白長さ：下
図枠と焼き付けるビューとの下側の余白長さ。
図枠がない場合は、無効です。
space=trueの場合のみ有効です。
初期値=5.0mm。
SxWF.project3D2D_2使用時のみ有効です。
mode_view=SxOpt3D2D_2.MODE_VIEW_AUTOの場合のみ有効です。
space_left 余白長さ：左
図枠と焼き付けるビューとの左側の余白長さ。
図枠がない場合は、無効です。
space=trueの場合のみ有効です。
初期値=5.0mm。
SxWF.project3D2D_2使用時のみ有効です。
mode_view=SxOpt3D2D_2.MODE_VIEW_AUTOの場合のみ有効です。
space_right 余白長さ：右
図枠と焼き付けるビューとの右側の余白長さ。
図枠がない場合は、ビュー間の横方向間隔です。
初期値=5.0mm。
space=trueの場合のみ有効です。
SxWF.project3D2D_2使用時のみ有効です。
mode_view=SxOpt3D2D_2.MODE_VIEW_AUTOの場合のみ有効です。
space_top 余白長さ：上
図枠と焼き付けるビューとの上側の余白長さ。
図枠がない場合は、ビュー間の縦方向間隔です。
初期値=5.0mm。
space=trueの場合のみ有効です。
SxWF.project3D2D_2使用時のみ有効です。
mode_view=SxOpt3D2D_2.MODE_VIEW_AUTOの場合のみ有効です。
tangent_angle 正接線許容誤差(度)。
初期値=0.0。
tangent_modeが、SxOpt3D2D_2.TANGENT_OFF以外の場合のみ有効です。
tangent_mode 正接線処理。
SxOpt3D2D_2.TANGENT_ON : 正接線を焼き付けます。(デフォルト)
SxOpt3D2D_2.TANGENT_OFF : 正接線を除去します。
SxOpt3D2D_2.TANGENT_DRAFT : 正接線を下書き線で焼き付けます。
view_back 焼付け対象ビュー：背面図(-XYビュー)
true:対象
false:非対象(デフォルト)
SxWF.project3D2D_2使用時のみ有効です。
view_bottom 焼付け対象ビュー：下面図(XZビュー)
true:対象
false:非対象(デフォルト)
SxWF.project3D2D_2使用時のみ有効です。
view_front 焼付け対象ビュー：正面図(XYビュー)
true:対象
false:非対象(デフォルト)
SxWF.project3D2D_2使用時のみ有効です。
view_left 焼付け対象ビュー：左側面図(ZYビュー)
true:対象
false:非対象(デフォルト)
SxWF.project3D2D_2使用時のみ有効です。
view_right 焼付け対象ビュー：右側面図(-ZYビュー)
true:対象
false:非対象(デフォルト)
SxWF.project3D2D_2使用時のみ有効です。
view_top 焼付け対象ビュー：平面図(X-Zビュー)
true:対象
false:非対象(デフォルト)
SxWF.project3D2D_2使用時のみ有効です。
