SxSys.getDistance メソッド 

2要素間の距離を測定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getDistance( _
   ByVal obj1 As Object, _
   ByVal obj2 As Object _
) As SxInfDist
[C#]
public static SxInfDist getDistance(
   object obj1,
   object obj2
);

パラメータ

obj1
測定対象要素1
obj2
測定対象要素2

戻り値

最短距離情報

解説

obj1, obj2には、以下のインスタンスが指定可能です。
SxPos(座標)
SxEdge(エッジ)
SxFace(フェイス)
SxEntSeg(要素：ワイヤ要素、立体、ソリッド、FCソリッド)
対象要素の属するVS又はWFがアクティブに設定されます。
2次元要素で、交点を持つ要素同士の場合は、処理できません。
立体要素(立体セグメント、ソリッド、FCソリッド)と、エッジ又はフェイス間の距離の測定はできません。
(この場合、エッジ/フェイスの属する要素と要素の距離を返します)

例外

例外の種類 条件
SxException CAD操作でのエラー情報
Exception その他の例外
