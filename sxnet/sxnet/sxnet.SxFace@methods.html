SxFace メソッド

SxFace クラスのメソッドは以下のとおりです。SxFace クラスの完全なメンバ一覧については、 SxFace メンバ のトピックを参照してください。

静的 パブリック (共有) メソッド

getColorList 指定フェイス群の色を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getEdgeListオーバーロード。 指定フェイス群に属するエッジを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getGeomList 指定フェイス群のジオメトリを一括取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getMassList 指定フェイス群ののマスプロパティを一括取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setColorオーバーロード。 フェイスの色を設定します。

パブリック メソッド

eval フェイスの指定点での法線を問い合わせます。
getCenterPoint フェイスの中心点を問い合わせます。
getColor フェイスの色を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getEdgeListオーバーロード。 フェイスに属するエッジを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getEntSeg フェイスの属する親セグメントを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getGeom フェイスのジオメトリを問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getMass フェイスの重心、面積を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
getOnPoint 指定座標に最も近いフェイス上の座標を問い合わせます。
isEqual 指定SxFaceインスタンスが自インスタンスとCAD上の同一フェイスを指しているか問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
isValid フェイスがCAD上の有効なフェイスを指しているか問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。
setColorオーバーロード。 フェイスの色を設定します。
