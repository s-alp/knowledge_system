SxVS.convExtent メソッド 

自VS上のエクステントを、グローバルビュー上のエクステントに変換します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function convExtent( _
   ByVal box As SxBox _
) As SxBox
[C#]
public SxBox convExtent(
   SxBox box
);

パラメータ

box
VSのエクステント

戻り値

グローバルビュー上でのエクステント

解説

自VSがビュー/ローカルビューでない場合は、boxの値をそのまま返答します。
ローカルビューが回転して配置されている場合、回転後のエクステントから、 グローバルビュー上のエクステントを再計算します。
