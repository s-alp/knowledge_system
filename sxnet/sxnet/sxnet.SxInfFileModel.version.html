SxInfFileModel.version フィールド

図面ファイル バージョン

[Visual Basic]
Public version As Integer
[C#]
public int version;

解説

図面ファイルの保存バージョン番号です。
例：5の場合、V5形式で保存された図面ファイルです。
3の場合、V3形式で保存された図面ファイルです。
0の場合、V2L9以前の図面ファイル、又は、.icd形式以外のファイルです。
