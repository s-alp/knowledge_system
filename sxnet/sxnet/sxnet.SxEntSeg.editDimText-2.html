解説

自要素が、幾何公差枠(SEGTYPE_GTOL)の場合の、文字編集を行います。
自要素は、アクティブウィンドウのアクティブVS/WFの要素である必要があります。
データム、形体指示要素の文字編集は、editTextメソッドを使用して下さい。

幾何公差枠の公差値を変更する。[C#] 

/* 幾何公差枠の公差値を変更する。*/
void EditText(SxEntSeg seg, string text)
{
    // セグメントのジオメトリを取得します。
    SxGeom geom = seg.getGeom();

    // 要素が公差枠要素の場合のみ、処理します。
    if ( geom.type == SxGeom.TYPE_GTOL ){
        SxGeomTol geom_tol = (SxGeomTol)geom;
        SxGeomTol.Frame frame = geom_tol.frame[0];
        frame.str_tolvalue = text; // 公差値文字を変更します。
        seg.editDimText(frame);
    }
}
