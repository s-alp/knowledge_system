SxGeomArrow メンバ

SxGeomArrow 概要

パブリック コンストラクタ

SxGeomArrow コンストラクタ インスタンスを初期化します。

パブリック フィールド

arrow_ang 端末記号の矢印片側開き角度
arrow_type 端末記号の種類
SxDimLineAtr.ATRDIM_ARROW_ONE: 片側矢印
SxDimLineAtr.ATRDIM_ARROW_BOTH: 両側矢印
SxDimLineAtr.ATRDIM_ARROW_DOT: 黒丸
arrow_width 端末記号の矢印の幅
color 図形の色
cs 平面の定義座標系情報。
2次元寸法線の場合無効です。
dot_diam 端末記号の黒丸の直径
pnt_num 構成点座標数(始点、折曲点、終点)
pntlist 矢印の構成点座標の配列
type (SxGeom から継承されます) ジオメトリタイプ
width 図形の線幅
SxSys.WIDTH_BOLD : 太線
SxSys.WIDTH_MIDDLE : 中線
SxSys.WIDTH_THIN : 細線
