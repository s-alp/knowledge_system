実像部品に要素を追加します。

[Visual Basic]

)
[C#]
public void add(
   SxEnt[] entlist
);

パラメータ

entlist
追加する要素の配列

解説

entlistには、自実像部品と同一VS内の要素を指定して下さい。
標準部品(写像部品)、配置子図をentlist内に指定する事はできません。
entlistに実像部品又はグループを指定した場合、実像部品又はグループは解除されます。

自要素が属するVSをアクティブに設定します。
自要素を表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
