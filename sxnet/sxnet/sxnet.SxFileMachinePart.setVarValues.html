SxFileMachinePart.setVarValues メソッド 

指定呼び名のパラメタ値を設定します。

[Visual Basic]
Public Sub setVarValues( _
   ByVal pattern As String, _
   ByVal values As Double() _
)
[C#]
public void setVarValues(
   string pattern,
   double[] values
);

パラメータ

pattern
呼び名
values
パラメタ値リスト

解説

指定呼び名のパラメタ値を設定します。 一度設定したパラメタ値は、インスタンスが消滅するまで有効です。
方程式が定義されたパラメタには、パラメタ値は登録されません。（何もしません）

例外

例外の種類 条件
SxException 指定した呼び名は存在しない、又は、パラメタの個数が異なります。

使用例

[C#] 

void sample(){
    SxPos pos = new SxPos(0.0e0, 0.0e0, 0.0e0);
    SxFileMachinePart file_part = 
        new SxFileMachinePart(SxFileMachinePart.getDirJis() + "\\3D\\01_ナット\\Ｔ溝ナット\\Ｔ溝ナット@正面.icd");

    // 呼び"M5"の変数値を取り出す
    double[] values = file_part.getVarValues("M5");
    
    // 変数d1の値を変更（※変数名一覧は、getVarNamesメソッドで取り出し可)
    values[1] = 10.0;
    
    // 変更した変数値を有効に設定する。
    file_part.setVarValues("M5",values);
    
    // 設定したパラメタ値で部品を原点に配置(回転なし)
    SxPart ent_part = (SxEntPart)file_part.open(SxFileMachinePart.MODE_PART, 
                                                pos, 0.0e0, 0.0e0, 0.0e0, true);
}
