SxDraftAtrDimForm フィールド

SxDraftAtrDimForm クラスのフィールドは以下のとおりです。SxDraftAtrDimForm クラスの完全なメンバ一覧については、 SxDraftAtrDimForm メンバ のトピックを参照してください。

パブリック フィールド

dimval_space 寸法線と寸法値の間隔
note_space 上段の線と文字の間隔（注記）
note_underline_len 注記形状の下線の長さ
prog_bend_width 標準引き出しの折曲幅（累進/金型寸法線）
prog_diameter 基点記号の直径（累進/金型寸法線）
prog_space 補助線と寸法値の間隔（累進/金型寸法線）
step_space 段間隔（直列/並列寸法線)
underline1_len 外向寸法線の下線の長さ
(長さ/径/面取り寸法線)
underline2_len 外側矢印の時の下線の長さ
