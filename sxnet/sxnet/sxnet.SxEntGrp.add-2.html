グループにメンバの追加を行います。(全要素)

[Visual Basic]
Overloads Public Sub add()
[C#]
public void add();

解説

自グループの属する、VS/WF内の全要素をグループに追加します。
パーツは対象外です。
外部パーツに属する要素は対象外です。
要素の属するVS又はWFがアクティブに設定されます。
要素を表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
