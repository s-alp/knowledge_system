SxDraftAtrGeneral 概要

パブリック フィールド

aidline_extlen 補助線突き出し長さ

arrow_ang 端末記号の矢印片側開き角度(度)

SxDimLineAtr.ATRDIM_ARROW_BOTH : 両側矢印
SxDimLineAtr.ATRDIM_ARROW_DOT : 黒丸
arrow_width 端末記号の矢印の幅
color 線色
dot_diam 端末記号の黒丸の直径
width 線幅
