SxEnt.getExtentList メソッド 

指定要素群のエクステント情報を一括取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getExtentList( _
   ByVal entlist As SxEnt(), _
   ByVal merge As Boolean _
) As SxBox()
[C#]
public static SxBox[] getExtentList(
   SxEnt[] entlist,
   bool merge
);

パラメータ

entlist
問合せを行う要素の配列
merge
エクステントの統合。
true:各エクステントを統合して、1つのエクステントを返します。
false:エクステントの統合を行わずに、要素毎のエクステントを返します。

戻り値

エクステントの配列

解説

指定要素群のエクステントボックスを問い合わせます。
エクステントは、要素を包含する正確な最小限のボックスである事は保証しません。
要素を包含するボックスの目安と考えて下さい。
非表示の要素、又は、エクステントを持たない要素の場合、該当する配列の要素に null(VBの場合Nothing)を設定して返答します。
3Dパーツの場合、子パーツも含むエクステントを返答します。
要素が、非表示レイヤ、非検索レイヤに属していても、エクステントを返します。

merge=trueの場合、entlistで指定された要素群全体のエクステントを1つ返します。
entlistには、同一VS又は同一WF内の要素を指定して下さい。
