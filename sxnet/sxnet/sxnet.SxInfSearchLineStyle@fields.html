SxInfSearchLineStyle フィールド

SxInfSearchLineStyle クラスのフィールドは以下のとおりです。SxInfSearchLineStyle クラスの完全なメンバ一覧については、 SxInfSearchLineStyle メンバ のトピックを参照してください。

パブリック フィールド

status 各線種の検索状態。(配列サイズ=7)
true:検索, false:非検索
[0] : システム予約
[SxSys.STYLE_SOLID] : 実線の検索状態
[SxSys.STYLE_DASH] : 破線の検索状態
[SxSys.STYLE_ONE_DOT_CHAIN] : 一点鎖線の検索状態
[SxSys.STYLE_TWO_DOT_CHAIN] : 二点鎖線の検索状態
[SxSys.STYLE_DOT] : 点線の検索状態
[SxSys.STYLE_LONG_DASH] : 長破線の検索状態
