SxInfPrint メンバ

SxInfPrint 概要

パブリック コンストラクタ

SxInfPrint オーバーロード。 SxInfPrint クラスの新しいインスタンスを初期化します。

パブリック フィールド

dinfo 用紙情報
[0]:用紙サイズ幅
[1]:用紙サイズ高さ
[2]:作画スケール(-1.0:自動)
[3]:作画範囲左下X座標
[4]:作画範囲左下Y座標
[5]:作画範囲右上X座標
[6]:作画範囲右上Y座標
[7]〜[19]:未使用(システム予約領域)
※座標は、2次元グローバルビュー上の座標値です。
no 出図範囲枠番号(1〜16)
size 用紙サイズ
"A0"〜"A6", "B1"〜"B7" : 規格サイズ
"XY" : 任意サイズ
vertical 用紙方向 true:縦 false:横
