SxVS.setOrigin メソッド 

ビュー/ローカルビューの原点を変更します。
子図の基準点を変更します。

[Visual Basic]
Public Sub setOrigin( _
   ByVal pos As SxPos, _
   ByVal mode As Boolean _
)
[C#]
public void setOrigin(
   SxPos pos,
   bool mode
);

パラメータ

pos
原点/基準点に設定する座標
mode
要素移動モード。
true:ビュー内の要素を原点位置に合わせて移動します。
false:ビュー座標軸を原点位置に移動します。
子図の場合は本パラメタは無視されます。

解説

ビュー原点位置の変更を行います。
posは、自VS座標系での座標値です。グローバルビュー座標系ではありません。
自VSを表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
自VSを表示するウィンドウがアクティブウィンドウに設定されます。
自VSがアクティブVS(アクティブビュー)になります。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
