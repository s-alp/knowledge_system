SxInfSelect フィールド

SxInfSelect クラスのフィールドは以下のとおりです。SxInfSelect クラスの完全なメンバ一覧については、 SxInfSelect メンバ のトピックを参照してください。

静的 パブリック (共有) フィールド

POS_STATUS_ANY 座標選択時の座標種別定数：任意の座標が選択されました。
POS_STATUS_EDGE 座標選択時の座標種別定数：エッジ上の座標が選択されました。
POS_STATUS_ENT 座標選択時の座標種別定数：要素上の座標が選択されました。
POS_STATUS_FACE 座標選択時の座標種別定数：フェイス上の座標が選択されました。
POS_STATUS_INT 座標選択時の座標種別定数：交点座標が選択されました。
STATUS_EDGE 実行結果種別定数：エッジが選択されました。
STATUS_ENT 実行結果種別定数：要素が選択されました。
STATUS_FACE 実行結果種別定数：フェイスが選択されました。
STATUS_GO 実行結果種別定数：@GO(マウス右ボタン)が選択されました。
STATUS_POS 実行結果種別定数：座標が選択されました。

パブリック フィールド

edge 選択されたエッジ。
status = SxInfSelect.STATUS_EDGEの場合に設定されます。
statusが上記以外の場合、null(VBの場合、Nothing)です。
ent 選択された要素。
status = SxInfSelect.STATUS_ENTの場合に設定されます。
statusが上記以外の場合、null(VBの場合、Nothing)です。
face 選択されたフェイス。
status = SxInfSelect.STATUS_FACEの場合に設定されます。
statusが上記以外の場合、null(VBの場合、Nothing)です。
hit_edge1 座標選択時の座標上のエッジ。
エッジ上の座標を指定した場合のみ有効です。
上記以外の場合、null(VBの場合、Nothing)です。
hit_edge2 交点選択時の座標上のエッジ2。
エッジ同士の交点座標を指定した場合のみ有効です。
上記以外の場合、null(VBの場合、Nothing)です。
セグメントとエッジの交点が選択された場合、hit_seg1,hit_edge1が設定されます。
hit_face1 座標選択時の座標上のフェイス。
フェイス上の座標を選択した場合のみ有効です。
上記以外の場合、null(VBの場合、Nothing)です。
hit_pos 要素選択時のCAD上でのマウスHIT座標。
要素/エッジ/フェイスを選択した場合のみ有効です。
上記以外の場合、null(VBの場合、Nothing)です。
hit_seg1 座標選択時の座標上のセグメント要素。
セグメント要素上の座標を指定した場合のみ有効です。
上記以外の場合、null(VBの場合、Nothing)です。
hit_seg2 交点選択時の座標上のセグメント要素2。
セグメント同士の交点座標を指定した場合のみ有効です。
上記以外の場合、null(VBの場合、Nothing)です。
セグメントとエッジの交点が選択された場合、hit_seg1,hit_edge1が設定されます。
pos 選択された座標。
status = SxInfSelect.STATUS_POSの場合に設定されます。
statusが上記以外の場合、null(VBの場合、Nothing)です。
pos_status 座標選択時の座標種別。
status=SxInfSelect.STATUS_POSの場合のみ有効です。

SxInfSelect.POS_STATUS_ANY : 任意座標
SxInfSelect.POS_STATUS_ENT : 要素上の座標
SxInfSelect.POS_STATUS_EDGE : エッジ上の座標
SxInfSelect.POS_STATUS_FACE : フェイス上の座標
SxInfSelect.POS_STATUS_INT : 交点座標
status 実行結果種別。
SxInfSelect.STATUS_GO : @GO(マウス右ボタン)が選択されました。
SxInfSelect.STATUS_ENT : 要素が選択されました。
SxInfSelect.STATUS_EDGE : 立体/FCソリッド/ソリッドのエッジが選択されました。
SxInfSelect.STATUS_FACE : 立体/FCソリッド/ソリッドのフェイスが選択されました。
SxInfSelect.STATUS_POS : 座標が選択されました。
