SxGeomCutLine 概要

パブリック コンストラクタ

SxGeomCutLine コンストラクタ インスタンスを初期化します。

パブリック フィールド

arrow 切断記号の矢印の有無。
true:矢印有り
false:矢印無し
atr_word 文字属性情報
cs 平面の定義座標系情報。
2次元寸法線の場合無効です。
name 切断名
pnt1 切断名1配置点座標
pnt2 切断名2配置点座標
type (SxGeom から継承されます) ジオメトリタイプ
