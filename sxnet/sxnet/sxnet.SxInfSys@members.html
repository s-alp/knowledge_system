SxInfSys メンバ

SxInfSys 概要

パブリック コンストラクタ

SxInfSys コンストラクタ 指定SxInfSysインスタンスのコピーを作成します。

パブリック フィールド

active_asmplane アクティブ組立平面。
アクティブウィンドウにVSが表示されている場合、または、アクティブ組立平面が
存在しない場合、null(VBの場合、Nothing)です。
active_layer 入力レイヤ番号
active_model アクティブモデル。
アクティブモデルが存在しない場合は、null(VBの場合、Nothing)です。
active_part アクティブWF内の入力パーツ。
入力パーツが存在しない場合は、null(VBの場合、Nothing)です。
active_wfがnullの場合も、nullです。
active_vs アクティブVS。
アクティブウィンドウにWFが表示されている場合、null(VBの場合、Nothing)です。
アクティブウィンドウにグローバルビューVSが表示されている場合、 アクティブビューVSを返します。
active_wf アクティブWF。
アクティブウィンドウにVSが表示されている場合、null(VBの場合、Nothing)です。
active_window アクティブウィンドウ。
アクティブウィンドウが存在しない場合は、null(VBの場合、Nothing)です。
cs アクティブウィンドウの入力座標系情報。
入力座標系が有効になっていない、又は絶対座標系と同じ場合は、null(VBの場合Nothing)です。
fix_plane_input 入力組立平面ロックフラグ
true:固定
false:解除
grid グリッド間隔
0.0の場合、グリッドなしです。
is_viewer ビューアフラグ
true: iCAD SX ビューア／iCAD SX 組立支援ツール
false: iCAD SX ビューア／iCAD SX 組立支援ツール以外のシステム
level システムレベル番号
models 開いているモデル一覧。
モデルが1つも開かれていない場合は、null(VBの場合、Nothing)です。
path CADシステムインストールフォルダパス
pen_color 要素色番号
pen_style 線種番号
pen_width 線幅番号
scale アクティブモデルの2次元図面尺度
sys3d CADシステム種別。
true:3Dシステム, false:2Dシステム
version システムバージョン番号
windows 開いているウィンドウ一覧。
ウィンドウが1つも開かれていない場合は、null(VBの場合、Nothing)です。
