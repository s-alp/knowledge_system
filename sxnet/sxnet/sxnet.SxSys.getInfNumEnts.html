SxSys.getInfNumEnts メソッド 

アクティブウィンドウに表示されているモデル(図面)の要素の数を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getInfNumEnts( _
   ByVal layer As Boolean, _
   ByVal allview As Boolean _
) As SxInfNumEnts()
[C#]
public static SxInfNumEnts[] getInfNumEnts(
   bool layer,
   bool allview
);

パラメータ

layer
処理種別:
true:レイヤ別
false:要素タイプ別
allview
全ビュー対象:
true:全ビューを対象にします。
false:アクティブビューを対象にします。
※アクティブウィンドウが2Dの場合のみ有効です。

戻り値

要素数情報インスタンスの配列

解説

アクティブウィンドウに表示されている、要素数の集計を行います。
要素が存在しない場合、null(VBの場合、Nothing)を返します。
レイヤ別の場合、SxInfNumEnts.ent_name = "" です。
要素タイプ別の場合、SxInfNumEnts.layer = 0です。
レイヤ別の場合、要素が存在しないレイヤの情報は返しません。

例外

例外の種類 条件
SxException CAD操作でのエラー情報
Exception その他の例外
