SxEntPart.getSamePartList メソッド 

同一パーツを問い合わせます。

[Visual Basic]
Public Function getSamePartList() As SxEntPart()
[C#]
public SxEntPart[] getSamePartList();

戻り値

自パーツを含むパーツインスタンスの配列

解説

自パーツを含む、同一パーツを問い合わせます。
自パーツが外部パーツの場合は同一外部パーツを返します。
自パーツが内部パーツの場合は同名内部パーツを返します。
配列の最初の要素は、自パーツです。
同一パーツが存在しない場合、自パーツのみを返します。
