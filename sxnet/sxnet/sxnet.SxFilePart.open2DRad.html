パラメータ

mode
処理モード
SxFilePart.MODE_SEG :実要素化
SxFilePart.MODE_RPART:実像部品化
SxFilePart.MODE_GRP :グループ化
SxFilePart.MODE_REFER:写像配置
pos
配置座標
scale
配置スケール
ang
配置角度(ラジアン)
layer
レイヤ引継ぎ(true:引き継ぐ, false:入力レイヤに作成)

戻り値

配置した部品
mode = SxFilePart.MODE_SEGの場合、nullを返します。(部品配置は実行されます)
mode = SxFilePart.MODE_RPARTの場合、SxEntRPartインスタンスを返します。
mode = SxFilePart.MODE_GRPの場合、SxEntGrpインスタンスを返します。
mode = SxFilePart.MODE_REFERの場合、SxEntReferインスタンスを返します。

解説

2次元標準部品を、アクティブウィンドウに表示された、アクティブVSに配置します。
mode=SxFilePart.MODE_REFERの場合、自インスタンスは部品フォルダ１〜３の部品ファイルでなければなりません。
部品フォルダは、SxFilePart.getPartDirメソッドで問い合わせ可能です。
