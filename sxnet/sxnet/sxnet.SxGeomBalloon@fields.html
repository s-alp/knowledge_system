パブリック フィールド

addball 追加風船のジオメトリ情報
atr_word 文字属性情報
auto_size 風船文字サイズ
true:自動
false:任意

diam 風船直径
lead_line 引出し線情報
lead_line_num 引出し線本数
line_color 風船の線色
line_width 風船の線幅

SxSys.WIDTH_MIDDLE : 中線
SxSys.WIDTH_THIN : 細線
n_addball 追加風船の数。
num_use 使用数(設定されていない場合は0)
pnt 基準風船配置座標
tbl 風船の段数
txt1 1段目の文字列
txt2 2段目の文字列。
tbl=2の場合のみ有効です。
type (SxGeom から継承されます) ジオメトリタイプ
