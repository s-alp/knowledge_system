この型の public static (Visual Basicでは Shared) のすべてのメンバは、 マルチスレッド操作で安全に使用できます。インスタンスのメンバの場合は、スレッドセーフであるとは限りません。

解説

寸法線の寸法値の属性情報を表すクラス。
各種寸法線のジオメトリ情報(SxGeom派生クラス)のメンバになります。
寸法線のジオメトリ情報の取得は、SxEntSegインスタンスの getGeomメソッドで行います。
寸法線の属性情報は、SxDimLineAtrクラスになります。
