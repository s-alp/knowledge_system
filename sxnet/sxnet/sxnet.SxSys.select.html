SxSys.select メソッド 

CAD上で要素/座標入力を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

パラメータ

opt
実行オプション

戻り値

CAD上での選択結果情報。
会話処理中に他のコマンドを選択する等して、 処理がキャンセルされた場合、null(VBの場合、Nothing)を返します。

解説

CAD上での会話操作で、指定した種別のオブジェクト(要素/エッジ/フェイス/座標)の選択処理 を行います。
選択するオブジェクトの種別の指定は、SxOptSelectで設定します。

選択結果をSxInfSelectのインスタンスとして返します。
CAD上での会話操作中に、他のコマンドを選択する等して、処理がキャンセルされた場合、 null(VBの場合、Nothing)を返します。

座標入力を行ったウィンドウに、アクティブな組立平面が存在し、かつ操作次元が2次元 の場合、座標は組立平面上の2次元座標系での値を返答します。

[C#]
CAD上で、要素又は座標の選択を実行する。
@GO(右ボタン)が押された場合、選択処理をキャンセルする。

void foo()
{
    SxOptSelect opt = new SxOptSelect();
    opt.select_ent = true;    // 要素の選択を有効にする。
    opt.select_pos = true;    // 座標の選択を有効にする。
    opt.select_go = true;    // @GO(右ボタン)での終了を有効にする。
    opt.ent_mode = SxSys.ENTMODE_SEG; // 要素の検索モードを、[要]にする。
    
    // CAD上での要素又は座標の選択の実行
    SxInfSelect inf = SxSys.select();
    
    // 選択結果の表示
    if ( inf == null )
    {
        Console.WriteLine("選択処理がキャンセルされました。");
    } 
    else 
    {
        if ( inf.status == SxInfSelect.STATUS_ENT )
        {
            Console.WriteLine("要素が選択されました。");
            Console,WriteLine("要素のタイプ=" + inf.ent.Type);
        }
        else if ( inf.status == SxInfSelect.STATUS_POS )
        {
            Console.WriteLine("座標が選択されました。");
            Console.WriteLine(
                string.Format("座標=({0},{1},{2})",inf.pos.x, inf.pos.y, inf.pos.z) );
        }
        else if ( inf.status == SxInfSelect.STATUS_GO )
        {
            Console.WriteLine("選択処理がキャンセルされました。");
        }
    }
}
