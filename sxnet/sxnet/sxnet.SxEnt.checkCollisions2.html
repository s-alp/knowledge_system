3次元要素の高速干渉チェックを行います。

   ByVal entlist As SxEnt(), _

public static SxInfCollision[] checkCollisions2(
   SxEnt[] entlist,

);

パラメータ

entlist
干渉チェックを行う要素配列
tol
クリアランス値
tol ＜＝ 0.0の場合、干渉チェックとします。
tol ＞ 0.0の場合、クリアランスチェックとします。
cg
干渉部分の形状の表示方法
true:CG表示
false:線画表示
※将来の予約パラメタ。現在は指定に関わらず必ずCG表示されます。
check_all
処理モード
true:要素群全ての干渉チェックを行います。(総当たり)
false:指定した要素周辺の干渉チェックを行います。(要素周辺)

戻り値

干渉チェック情報配列
干渉が存在しない場合、null(VBの場合、Nothing)を返答します。

解説

指定された複数3次元要素の高速干渉チェックを行います。
tol値はクリアランスチェックの場合のみ有効です。
entlistには、アクティブウィンドウに表示されている、同一WF内の要素を指定して下さい。
干渉部分の形状を環境セットアップで定義された色で作成します。

例外
