SxOptLengthDim メンバ

SxOptLengthDim 概要

静的 パブリック (共有) フィールド

DIR_ELEMENT 寸法線方向定数：要素方向
DIR_HORIZON 寸法線方向定数：水平方向
DIR_VERTICAL 寸法線方向定数：垂直方向

パブリック コンストラクタ

SxOptLengthDim コンストラクタ 長さ寸法線作成オプションクラスで、インスタンスを初期化します。

パブリック フィールド

autocorrect 自動配置補正
true: 自動補正ON
false: 自動補正OFF(デフォルト)
autocorrect（自動配置補正）は直列、累進、金型のみ有効
autoword 文字任意位置
true: 任意位置ON（配置点位置に寸法値を配置します）
false: 任意位置OFF（配置点位置を参考しません）(デフォルト)
autoword（文字任意位置）は標準、中心線のみ有効
basenm 基準名
省略時は空文字列""を指定して下さい。(デフォルト)
direction 寸法線方向
SxOptLengthDim.DIR_ELEMENT: 要素方向(デフォルト)
SxOptLengthDim.DIR_HORIZON: 水平方向
SxOptLengthDim.DIR_VERTICAL: 垂直方向
direction（寸法線方向）は標準、直列、並列、金型のみ有効。
disp_zero 0表示or非表示
true: 0表示
false: 非表示(デフォルト)
disp_zero（0表示or非表示）は累進（縦表記場合）、金型のみ有効
down_tolword 下段公差値文字
省略時は空文字列""を指定して下さい。(デフォルト)
follow_terminalmark 端末記号追従
true: 端末記号追従ON
false: 端末記号追従OFF(デフォルト)
follow_terminalmark（端末記号追従）はは標準、中心線のみ有効
mark 記号情報
SxDimValueAtr.ATRDIM_MARK_NONE : 無し(デフォルト)
SxDimValueAtr.ATRDIM_MARK3_FAIMK : φ記号
SxDimValueAtr.ATRDIM_MARK3_SQMK : □記号
SxDimValueAtr.ATRDIM_MARK3_RMK : Ｒ記号
SxDimValueAtr.ATRDIM_MARK3_TMK : ｔ記号
SxDimValueAtr.ATRDIM_MARK3_MMK : Ｍ記号
SxDimValueAtr.ATRDIM_MARK3_OPT : 任意記号
opt_word 任意記号文字列
mark = SxDimValueAtr.ATRDIM_MARK3_OPTの場合のみ有効です。
(文字列"任意記号"デフォルト)
orthographic 縦表記or横表記
true: 縦表記
false: 横表記(デフォルト)
orthographic（縦表記or横表記）は累進のみ有効
steptype 2段or1段
true: 2段
false: 1段(デフォルト)
steptype（2段or1段）は直列のみ有効
upper_tolword 上段公差値文字
] 省略時は空文字列""を指定して下さい。(デフォルト)
