SxInfPart フィールド

SxInfPart クラスのフィールドは以下のとおりです。SxInfPart クラスの完全なメンバ一覧については、 SxInfPart メンバ のトピックを参照してください。

パブリック フィールド

comment コメント
未設定の場合、空文字""です。
cs パーツ定義座標情報
is_external 外部パーツ情報。
true:外部パーツ
false:内部パーツ
is_mirror ミラー情報
true:ミラーされています。
false:ミラーされていません。
is_read_only 外部パーツのアクセス権。
true:読み取り専用
false:書き込み可能
※自図面内の内部パーツの場合、常にfalseです。
is_unloaded 未解決パーツ情報。
true:未解決パーツ。
false:未解決パーツではありません。
※未解決パーツ:参照先図面が未解決の外部パーツです。
name パーツ名
path 参照図面格納フォルダ。
外部パーツ、又は、外部パーツ配下の内部パーツの場合のみ有効です。
ref_model_name パーツ参照図面名。
外部パーツ、又は、外部パーツ配下の内部パーツの場合のみ有効です。
