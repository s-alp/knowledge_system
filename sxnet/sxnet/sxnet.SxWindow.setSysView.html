SxWindow.setSysView メソッド 

3次元システムビュー表示を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Sub setSysView( _
   ByVal view As Integer _
)
[C#]
public void setSysView(
   int view
);

パラメータ

view
システムビュー番号。
SxWindow.VIEW_TOP：平面図
SxWindow.VIEW_FRONT：正面図
SxWindow.VIEW_RIGHT：右側面図
SxWindow.VIEW_LEFT：左側面図
SxWindow.VIEW_BACK：背面図
SxWindow.VIEW_BOTTOM：下面図

解説

3次元システムビュー表示を行います。
ウィンドウが3次元の場合のみ使用可能です。
