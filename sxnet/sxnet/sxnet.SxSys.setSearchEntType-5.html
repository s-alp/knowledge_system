SxSys.setSearchEntType メソッド (Int32, Boolean)

検索要素タイプを設定します。(全検索/全非検索)
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

)
[C#]

   int mode,
   bool onoff
);

パラメータ

mode
検索タイプを設定する要素種別。
SxSys.SEARCH_ENT_TYPE_ALL : 全要素(作図要素、製図要素、立体要素)
SxSys.SEARCH_ENT_TYPE_DRAW : 作図要素
SxSys.SEARCH_ENT_TYPE_DRAFT : 製図要素
SxSys.SEARCH_ENT_TYPE_SOLID : 立体要素
onoff
検索属性。
true:検索
false:非検索

解説

CAD上の検索要素タイプの設定を行います。
mode=SxSys.SEARCH_ENT_TYPE_ALLの場合、全ての作図要素、製図要素、立体要素の検索タイプを、 onoffで指定された属性に設定します。
mode=SxSys.SEARCH_ENT_TYPE_DRAWの場合、全ての作図要素の検索タイプを、 onoffで指定された属性に設定します。
mode=SxSys.SEARCH_ENT_TYPE_DRAFTの場合、全ての製図要素の検索タイプを、 onoffで指定された属性に設定します。
mode=SxSys.SEARCH_ENT_TYPE_SOLIDの場合、全ての立体要素の検索タイプを、 onoffで指定された属性に設定します。
