SxVS.getEntList メソッド 

要素の矩形検索を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function getEntList( _
   ByVal p1 As SxPos, _
   ByVal p2 As SxPos, _
   ByVal part As Boolean, _
   ByVal cross As Boolean _
) As SxEnt()
[C#]
public SxEnt[] getEntList(
   SxPos p1,
   SxPos p2,
   bool part,
   bool cross
);

パラメータ

p1
検索矩形対角座標1
p2
検索矩形対角座標2
part
実像部品検索。
true:実像部品に属するセグメント要素を検索した場合、実像部品を返答します。
false:実像部品に属するセグメント要素も、セグメントとして返答します。
cross
矩形線上検索。 true:矩形境界に干渉する要素も検索します。
false:矩形に完全に包含された要素のみ検索します。

戻り値

検索された要素の配列

解説

自VS内の要素の矩形検索を行います。
非表示の要素、表示レイヤ、非検索レイヤ、非検索タイプの要素は、検索対象外です。
検索要素が何もない場合、null(VBの場合、Nothing)を返します。
自VSをアクティブPDに表示し、アクティブVSに設定します。
