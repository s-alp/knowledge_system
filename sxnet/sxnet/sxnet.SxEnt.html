スレッドセーフ

この型の public static (Visual Basicでは Shared) のすべてのメンバは、 マルチスレッド操作で安全に使用できます。インスタンスのメンバの場合は、スレッドセーフであるとは限りません。

解説

CAD上の要素(セグメント、グループ、レファー、3Dパーツ、2D実像部品)の基本抽象クラスです。
インスタンスは、本クラスから派生したクラスで生成します。
移動、削除、コピー等の要素共通の編集処理、情報取得処理を行う事ができます。

セグメント(SxEntSegインスタンス)の創成については、SxEntSegクラスを参照して下さい。
グループ(SxEntGrpインスタンス)の創成については、SxEntGrpクラスを参照して下さい。
3Dパーツ(SxEntPartインスタンス)の創成については、SxEntPartクラスを参照して下さい。
2D実像部品(SxEntRPartインスタンス)の創成については、SxEntRPartクラスを参照して下さい。
写像部品/配置子図(SxEntReferインスタンス)の創成については、SxFilePartクラス、 SxEntReferクラスを参照して下さい。

VS(2次元図面空間)内の各種要素の取得については、SxVSクラスを参照して下さい。
WF(3次元図面空間)内の各種要素の取得については、SxWFクラスを参照して下さい。

CAD上から要素を選択して取得する方法については、SxSysクラスの、 getEntメソッド、getEntListメソッド、 selectメソッドを参照して下さい。
