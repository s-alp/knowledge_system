SxSys.setDispLayer メソッド (Int32, Boolean)

アクティブモデルのレイヤの表示/非表示の設定を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Shared Sub setDispLayer( _
   ByVal layer As Integer, _
   ByVal mode As Boolean _
)
[C#]
public static void setDispLayer(
   int layer,
   bool mode
);

パラメータ

layer
表示/非表示の設定を行うレイヤ番号。(1〜255)
mode
表示属性。
true:表示
false:非表示

解説

アクティブウィンドウに表示中のモデルの表示レイヤの設定を行います。
mode=falseの場合、検索レイヤも合わせて非検索に設定します。
入力レイヤを非表示に指定した場合は、無視されます。(入力レイヤは常に表示・検索です)
