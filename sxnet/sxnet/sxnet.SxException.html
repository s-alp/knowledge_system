スレッドセーフ

この型の public static (Visual Basicでは Shared) のすべてのメンバは、 マルチスレッド操作で安全に使用できます。インスタンスのメンバの場合は、スレッドセーフであるとは限りません。

解説

CAD操作中に発生したエラー情報の例外クラスです。
Messageプロパティにエラーメッセージを格納します。

使用例

エラーメッセージをコンソールに出力する。 

[C#]
using System;
using sxnet;
class drawLine()
{
    static void Main(string[] args){
        try{
            // 同一座標を指定して、線分セグメントを作成.
            SxPos p1 = new SxPos(0.0e0, 0.0e0, 0.0e0);
            SxEntSeg.createLine3D( p1, p1 );
        }catch(SxException e){
            // エラーメッセージをコンソールに出力する.
            Console.WriteLine(e.Message);
        }
    }
}
実行結果:
MSG00115 座標，要素が同一，または近すぎます。( 1101)
