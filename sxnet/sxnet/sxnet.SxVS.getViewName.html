SxVS.getViewName メソッド 

基本ビュータイプから対応するビュー名を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getViewName( _

) As String
[C#]

   int viewtype
);

パラメータ

viewtype
基本ビュータイプ。
SxVS.VIEW_FRONT : 正面図
SxVS.VIEW_BACK : 背面図
SxVS.VIEW_TOP : 平面図
SxVS.VIEW_BOTTOM : 下面図
SxVS.VIEW_LEFT : 左側面図
SxVS.VIEW_RIGHT : 右側面図

戻り値

指定基本ビューのビュー名(VS名)

解説

指定された基本ビューのタイプから、対応するビュー名を問い合わせます。
