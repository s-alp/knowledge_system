SxInf3DParam メンバ

SxInf3DParam 概要

静的 パブリック (共有) フィールド

TYPE_ANGLE パラメタタイプ定数：角度
TYPE_DIAMETER パラメタタイプ定数：直径
TYPE_LENGTH パラメタタイプ定数：長さ
TYPE_NUMBER パラメタタイプ定数：数値
TYPE_POS パラメタタイプ定数：座標
TYPE_RADIUS パラメタタイプ定数：半径

静的 パブリック (共有) メソッド

getVarIndex 指定パラメタ名を持つSxInf3DParamインスタンスの配列内インデックス番号を問い合わせます。

パブリック コンストラクタ

SxInf3DParam コンストラクタ 指定SxInf3DParamインスタンスのコピーを作成します。

パブリック フィールド

dvalue パラメタ設定値(角度の場合、単位は度)
TypeがTYPE_POS以外の場合のみ有効です。
pos パラメタ設定値(座標値)
TypeがTYPE_POSの場合のみ有効です。
TYPE_POS以外の場合、null(VBの場合、Nothing)です。
status パラメタの状態。
true:有効
false:無効

パブリック プロパティ

Comment 条件名
条件名が設定されていない場合、空文字列""を返します。
Equation 本パラメタに定義されている方程式。
式が定義されていない場合、空文字列""を返します。
Explain 方程式の説明文。定義されていない場合、空文字列""を返します。
Name パラメタ名
Type パラメタタイプ
タイプ定数値については、パラメタタイプ定数(SxInf3DParam.TYPE_xxx)を参照して下さい。
