SxEntPart.getInfExList メソッド 

指定パーツ群の任意情報を一括取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getInfExList( _
   ByVal partlist As SxEntPart() _
) As String()
[C#]
public static string[] getInfExList(
   SxEntPart[] partlist
);

パラメータ

partlist
問合せを行うパーツの配列

戻り値

パーツの任意情報文字列の配列

解説

指定パーツ群に設定された任意情報文字列を一括して問い合わせます。
partlistには、同一モデル内のパーツを指定して下さい。
パーツが任意情報を持たない場合、 返答配列内の文字列にはnull(VBの場合Nothing)を設定して返します。
例えば、PARTS-A,PARTS-Bの2つを指定して、問い合わせた場合に、 PARTS-Aには任意情報が設定されておらず、PARTS-Bには設定されている場合、 返答される文字列の配列は、以下のようになります。
文字列配列[0] = null (VBの場合、Nothing)
文字列配列[1] = PARTS-Bの任意情報文字列。(CSV形式)
