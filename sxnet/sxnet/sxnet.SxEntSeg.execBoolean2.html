パラメータ

entlist
ツール要素の配列
mode
集合演算の種別
SxEntSeg.BOOL_UNITE : 和の演算処理を行う
SxEntSeg.BOOL_SUBTRACT : 差の演算処理を行う
SxEntSeg.BOOL_INTERSECT : 積の演算処理を行う
erase
ツール要素の削除
true:削除します。
false:削除しません。
※mode = SxEntSeg.BOOL_UNITEの時は、本パラメタの指定に関係なく、無条件に削除されます。

戻り値

集合演算結果情報

解説

自要素をターゲット要素として、指定ツール要素との集合演算を行います。
自要素、ツール要素は、立体、ソリッド、FCソリッドである必要があります。
mode=SxEntSeg.BOOL_INTERSECTかつ、erase=falseの場合、 自要素はそのまま残り、新しく作成されたFCソリッドを返答します。
上記の条件以外の場合：
自要素がFCソリッドの場合、自要素に対して直接演算を実行する為、自要素を返答します。
自要素がソリッド、立体セグメントの場合、新規に作成されたFCソリッドを返答します。 この場合、自要素も新しく作成されたFCソリッドに変わります。
非表示要素は処理対象外です。
