SxEntSeg.createSpun メソッド (SxPos, SxVec, SxVec, SxCoord[])

回転体セグメントを作成します。360度回転。
指定した閉ループを、回転軸周りに360度回転した立体形状を生成します。

パラメータ

org
回転体の定義座標系の原点座標
(回転軸原点位置)
zvec
回転体の定義座標系Z軸ベクトル
(回転軸ベクトル。回転体基準平面のY軸ベクトル)
xvec
回転体の定義座標系X軸ベクトル
(回転体基準平面のX軸ベクトル)
pnt_array
閉ループ座標情報の配列
定義座標系のXZ平面での2D座標を(x,z,0)の2D座標系で指定して下さい。
1点目は閉ループの開始座標であり、必ず、pnt_array[0].is_arc=falseに設定して下さい。
閉ループの座標は、XZ平面の第1、第4象限の座標(x>=0.0)を指定して下さい。
(閉ループが軸と交差してはいけません)

戻り値

作成した回転体セグメント

解説

アクティブウィンドウに表示されたWF上に、 pnt_array[]で指定されたXZ平面の面(閉ループ)を回転軸周りに360度回転した立体形状を作成します。
zvec,xvecは必ず直交している必要があります。
pnt_array[]は、定義座標系のXZ平面座標系(2D座標)で表された、閉ループを指定して下さい。
閉ループの自己交差は認められません。
又、始終点間以外の隣り合う座標間の距離は、0.0001以上離れている必要があります。
閉ループが回転軸から離れている場合は、外形の回転体から内形の回転体の差をとったFCソリッド を作成します。
閉ループの座標は、XZ平面の第1、第4象限の座標を指定して下さい。
(X座標に負の値は指定できません)
回転体セグメントで表現できない場合(端点数が32を超える場合、Y座標が単調増加でない場合等)は、 ソリッドで作成されます。

円柱形状を作成する。[C#]

// 回転体作成サンプル：円柱形状を作成する。

SxCoord[] loop1 = new SxCoord[5]; // 基準面ループ情報

// 回転面のループを定義する。
loop1[0] = new SxCoord( 0.0,  0.0);
loop1[1] = new SxCoord(10.0,  0.0);
loop1[2] = new SxCoord(10.0, 10.0);
loop1[3] = new SxCoord( 0.0, 10.0);
loop1[4] = new SxCoord(loop1[0]);

// 配置座標の指定(原点に配置)
SxPos org = new SxPos( 0.0, 0.0, 0.0 );

// 配置方向の指定(X軸方向が高さ方向となるよう回転)
SxVec zvec = new SxVec( 1.0, 0.0, 0.0 );
SxVec xvec = new SxVec( 0.0, 0.0, 1.0 );

// 回転体作成
SxEntSeg seg = SxEntSeg.createSpun( org, zvec, xvec, loop1 );
