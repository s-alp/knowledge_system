SxEntPart.add メソッド 

自パーツにメンバ要素の追加を行います。

[Visual Basic]
Public Sub add( _
   ByVal entlist As SxEnt() _
)
[C#]
public void add(
   SxEnt[] entlist
);

パラメータ

entlist
追加する要素の配列。
SxEntSeg又はSxEntPartインスタンスの配列を指定して下さい。
両者を混在させる事はできません。

解説

指定した要素群を自パーツに追加します。
entlistには、SxEntSeg又はSxEntPartのインスタンスの配列を指定して下さい。
両者を混在させる事はできません。
entlistに、別のパーツに属する要素を指定した時に、結果として別パーツの要素が全てなくなる場合、 そのパーツは削除されます。
自パーツの属するWF以外の要素をentlistに指定する事はできません。
外部パーツ内の要素を、他のパーツに追加した場合、又は、外部パーツに要素を追加した場合、 元の要素は削除されるため、追加後の、entlist[]内のインスタンスは、無効になります。
(CAD上での要素の追加処理は正常に行われます)
