SX.Net Class Library

SxSys.cancel メソッド 

実行中の処理をキャンセルし、プログラムに制御を戻します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Sub cancel()
[C#]
public static void cancel();

解説

実行中のメソッドをキャンセルし、プログラムに制御を戻します。
SxSys.getEnt()等の、会話処理のメソッドを強制キャンセルしたい場合に使用します。
SxSysクラスの、以下のメソッドが対象です。
getEnt, getEntList, getEdge, getEdgeList, getFace, getFaceList, getPos, getPosList
上記以外のメソッド呼び出し時に、本メソッドを使用した場合の動作は保証されません。
キャンセルされたメソッドの呼び出し元に、SxExceptionクラスの例外が返ります。
【注意】キャンセルが完了するまで（例外が返るまで）、次のメソッドを実行しないで下さい。
別スレッドで、メソッドが実行中でない場合は、何も処理を行いません。

[C#] 

 using System;
 using System.Threading;
 using sxnet;

 // SxSys.getPosList()メソッドを強制キャンセルする
 class Class1
 {
        [STAThread]
        static void Main(string[] args)
        {
            try
            {
                // システム初期化
                SxSys.init(3999);
    
                // スレッドで実行するオブジェクトの生成
                GetPosList getposlist = new GetPosList();
 
                // スレッドの生成
                Thread th1 = new Thread( new ThreadStart(getposlist.Run) );

                // SxSys.getPosListメソッドをスレッドで実行する。
                th1.Start();
                // 3秒待つ
                th1.Join(3000);

                // スレッドで実行中のSxSys.getPosListメソッドをキャンセルする。
                Console.WriteLine("SxSys.cancel() is called.");
                SxSys.cancel();

                // スレッド1の終了を待つ
                th1.Join();

            }
            catch( Exception e )
            {
                Console.WriteLine(e.Message);
            }
        }

        class GetPosList
        {
            public GetPosList(){;}

            public void Run()
            {
                try
                {
                    // SxSys.getPosList()メソッドの実行
                    Console.WriteLine("SxSys.getPosList() is executing.");
                    SxPos[] plist = SxSys.getPosList();
                }
                catch( SxException e )
                {
                    Console.WriteLine("{0} ircode = {1}-{2}-{3}", 
                        e.Message, e.ircode[0], e.ircode[1], e.ircode[2] );
                }
                catch( Exception e )
                {
                    Console.WriteLine(e.Message);
                }
            }
        }
    }

[実行結果] 
SxSys.getPosList() is executing.
SxSys.cancel() is called.
   プログラムから強制キャンセルされました。 ircode = 4-9999-1
