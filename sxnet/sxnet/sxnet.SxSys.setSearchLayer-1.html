SxSys.setSearchLayer メソッド (Int32, Boolean)

アクティブモデルのレイヤの検索/非検索の設定を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Shared Sub setSearchLayer( _
   ByVal layer As Integer, _
   ByVal mode As Boolean _
)
[C#]
public static void setSearchLayer(
   int layer,
   bool mode
);

パラメータ

layer
検索/非検索の設定を行うレイヤ番号。(1〜255)
mode
検索属性。
true:検索。
false:非検索。

解説

アクティブウィンドウに表示中のモデルの検索レイヤの設定を行います。
mode=trueの場合、表示レイヤも合わせて表示属性に設定されます。
入力レイヤを非検索に指定した場合、無視されます。(入力レイヤは常に表示・検索です)
