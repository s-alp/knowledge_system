最上位パーツのレイアウト変更を行います。

[Visual Basic]
Public Sub setCoordTopPart( _
   ByVal pos As SxPos, _
   ByVal zvec As SxVec, _
   ByVal xvec As SxVec _
)
[C#]
public void setCoordTopPart(
   SxPos pos,
   SxVec zvec,
   SxVec xvec
);

パラメータ

pos
原点座標
null(VBの場合Nothing)の場合、Z軸,X軸ベクトルのみ設定します。
zvec
Z軸単位ベクトル
nullの場合、原点座標のみ設定します。
xvec
X軸単位ベクトル
nullの場合、原点座標のみ設定します。

解説

自WF内の最上位パーツの、原点座標とZ軸,X軸ベクトルの変更を行います。
zvec, xvecは、直交関係にある必要があります。
自WFをアクティブWFに設定します。
