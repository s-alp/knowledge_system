SxSys.getAngle メソッド 

2要素間の角度を測定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Shared Function getAngle( _
   ByVal obj1 As Object, _
   ByVal obj2 As Object _
) As Double
[C#]
public static double getAngle(
   object obj1,
   object obj2
);

パラメータ

obj1
対象要素1
obj2
対象要素2

戻り値

角度(度)

解説

obj1,obj2に指定できるインスタンスは以下の通りです。
SxEntSeg(ワイヤ要素), SxEdge, SxFace
ワイヤ要素の場合は直線のみ指定可能です。
エッジの場合は直線のみ指定可能です。
フェイスの場合は平面のみ指定可能です。
それ以外の要素は処理対象外です。
面を指定する場合、obj1,obj2共に、面を指定して下さい。
要素の属するVS又はWFがアクティブに設定されます。
