SxEntSeg.editBalloonNum メソッド (Int32, Int32)

風船の使用数の設定/削除を行います。(風船番号指定)

[Visual Basic]
Overloads Public Sub editBalloonNum( _
   ByVal no As Integer, _
   ByVal num As Integer _
)
[C#]
public void editBalloonNum(
   int no,
   int num
);

パラメータ

no
使用数を編集する風船番号(0:基準風船、1以降:追加風船)
num
設定する使用数。0を指定した場合、使用数を削除します。

解説

自風船に対して、使用数の設定又は削除を行います。
自要素が風船セグメントである必要があります。
自要素は、アクティブウィンドウのアクティブVS/WFの要素である必要があります。

noには、編集を行う風船の番号を指定して下さい。0が基準風船、1以降は追加風船です。
numには、0〜9999までの数が指定可能です。0を指定した場合は、使用数の削除を行います。
