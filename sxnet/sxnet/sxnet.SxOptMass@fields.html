SxOptMass フィールド

SxOptMass クラスのフィールドは以下のとおりです。SxOptMass クラスの完全なメンバ一覧については、 SxOptMass メンバ のトピックを参照してください。

静的 パブリック (共有) フィールド

ACCURACY_HIGH 計算精度定数：高
ACCURACY_LOW 計算精度定数：低
ACCURACY_MIDDLE 計算精度定数：中

パブリック フィールド

axis_f 回転軸指定。
true:有り。
false:無し。(デフォルト)
※trueの場合、pos,vecを指定して下さい。
density 比重。
デフォルト値=1.0。
対象要素に材質情報が設定されている場合、本パラメタは無視されます。
is_SI SI単位系/工学単位系指定。
true:SI単位系
false:工学単位系(デフォルト)
mode_accuracy 自由曲面の計算精度。
SxOptMass.ACCURACY_LOW : 低(計算速度優先)(デフォルト)
SxOptMass.ACCURACY_MIDDLE : 中
SxOptMass.ACCURACY_HIGH : 高(計算精度優先)
※highの場合、処理時間がかかる場合があります。
mode_create_point CAD上での重心座標作成モード。
true:作成します。
false:作成しません。(デフォルト)
pos 回転軸上の座標。デフォルト値(0,0,0)
axis_f=trueの場合のみ有効です。
unit_type 単位。
SxInfMass.UNIT_MM_G:mm-g(デフォルト)
SxInfMass.UNIT_MM_KG:mm-kg
SxInfMass.UNIT_CM_G:cm-g
SxInfMass.UNIT_CM_KG:cm-kg
SxInfMass.UNIT_M_G:m-g
SxInfMass.UNIT_M_KG:m-kg
vec 回転軸ベクトル。デフォルト値(0,0,1)
axis_f=trueの場合のみ有効です。
