SxEntPart.takeOut メソッド 

内部パーツの切り出し(外部パーツ化)を行います。

[Visual Basic]
Public Sub takeOut( _
   ByVal all As Boolean _
)
[C#]
public void takeOut(
   bool all
);

パラメータ

all
同名内部パーツの一括切り出し
true:一括して切り出します。
false:自パーツのみ切り出します。

解説

自内部パーツを外部パーツに変換します。
自パーツが内部パーツである必要があります。
自パーツの属するモデルが保存済みである必要があります。
切り出した外部パーツのファイル名は、パーツ名と同じになります。

自パーツが属するWFをアクティブに設定します。
入力パーツは、WFの最上位に変更されます。
自パーツを表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
