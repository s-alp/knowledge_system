SxWF.getTopSegList メソッド (Int32, Int32, Boolean, Boolean, Boolean)

自WF内の最上位パーツ直下のセグメントを取得します。
(パーツに属していないセグメントの取得)
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

パラメータ

offset
読み込み開始オフセット(先頭=0)
num
最大読み込み個数。
0指定で、offset番目から全ての要素を返答します。
visi
非表示セグメントの処理。
true:返答する。
false:返答しない。
layer
非検索レイヤ上の要素の処理。
true:返答する。
false:返答しない。
type
非検索タイプの要素の処理。
true:返答する。
false:返答しない。

戻り値

セグメントの配列
返答する要素が存在しない場合、null(VBの場合、Nothing)を返答します。
自WFが3Dグローバルでない場合も、nullを返答します。

解説

自WF内の最上位のセグメントを取得します。
(パーツに属していないセグメントの取得)
返答する要素が存在しない場合、null(VBの場合、Nothing)を返答します。
自WFが3Dグローバルでない場合も、nullを返答します。
