SxEntRPart.setName メソッド 

実像部品名、コメントを設定します。

[Visual Basic]
Public Sub setName( _
   ByVal name As String, _
   ByVal comment As String, _
   ByVal change_all As Boolean _
)
[C#]
public void setName(
   string name,
   string comment,
   bool change_all
);

パラメータ

name
実像部品名
comment
実像部品コメント
省略時は空文字列""を指定して下さい。
change_all
同一名実像部品の一括処理
true:一括処理します。
false:一括処理を行いません。

解説

自実像部品の名前、コメント文字列を設定します。
実像部品名は、半角英数字で最大24文字までです。(全角文字で12文字まで)
コメントは、半角英数字で最大48文字までです。(全角文字で24文字まで)

自要素が属するVSがアクティブに設定されます。
自要素を表示するウィンドウが存在しない場合、新規にウィンドウを開設します。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
