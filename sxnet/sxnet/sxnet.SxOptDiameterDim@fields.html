SX.Net Class Library

SxOptDiameterDim フィールド

SxOptDiameterDim クラスのフィールドは以下のとおりです。SxOptDiameterDim クラスの完全なメンバ一覧については、 SxOptDiameterDim メンバ のトピックを参照してください。

静的 パブリック (共有) フィールド

DIR_ELEMENT 寸法線方向定数：要素方向
DIR_HORIZON 寸法線方向定数：水平方向
DIR_VERTICAL 寸法線方向定数：垂直方向

パブリック フィールド

angle 角度(度)
setangle=trueの場合のみ有効。
center_pass 中心通過。
true: 中心通過ON(デフォルト)
false: 中心通過OFF
center_pass（中心通過）は半径／標準のみ有効。
direction 寸法線方向
SxOptDiameterDim.DIR_ELEMENT: 要素方向(デフォルト)
SxOptDiameterDim.DIR_HORIZON: 水平方向
SxOptDiameterDim.DIR_VERTICAL: 垂直方向
direction（寸法線方向）は直径／２点間のみ有効
disp 寸法値表示。
true: 寸法値を表示します。(デフォルト)
false: 寸法値を表示しません。
disp（寸法値表示or非表示）は半径／（標準、稲妻、引出、寸法線無、微小1、微小2）のみ有効。
down_tolword 下段公差値文字
省略時は空文字列""を指定して下さい。(デフォルト)
follow_terminalmark 端末記号追従。
true: 端末記号追従ON
false: 端末記号追従OFF(デフォルト)
follow_terminalmark（端末記号追従）は直径／両側、直径／２点間、直径／中心線、
半径／中心線のみ有効。
lead_left 引出方向。
true: 左(デフォルト)
false: 右
lead_left(引出方向)は直径／引出、半径／引出、半径／微小１のみ有効
mark1 記号情報
SxDimValueAtr.ATRDIM_MARK_NONE : 無し(デフォルト)
SxDimValueAtr.ATRDIM_MARK3_FAIMK: φ記号(直径時特有)
SxDimValueAtr.ATRDIM_MARK3_RMK: Ｒ記号(半径時特有)
SxDimValueAtr.ATRDIM_MARK3_MMK: Ｍ記号
SxDimValueAtr.ATRDIM_MARK3_OPT: 任意記号
mark2 Ｓ・球記号情報。
SxDimValueAtr.ATRDIM_MARK_NONE : 無し(デフォルト)
SxDimValueAtr.ATRDIM_MARK2_SMK : Ｓ記号
SxDimValueAtr.ATRDIM_MARK2_QMK : 球記号
直径時mark1=SxDimValueAtr.ATRDIM_MARK3_FAIMKの場合のみ有効です。
半径時mark1=SxDimValueAtr.ATRDIM_MARK3_RMKの場合のみ有効です。
opt_word 任意記号文字列。
mark = SxDimValueAtr.ATRDIM_MARK3_OPTの場合のみ有効です。
省略時は空文字列""を指定して下さい。(デフォルト)
setangle 角度指定
true: 角度指定あり。
false：角度指定なし。(デフォルト)
setangle（角度指定）は直径／両側、直径／片側、直径／引出、半径／標準、半径／引出、
半径／微小1、半径／微小2のみ有効
str_num 個数文字。
省略時は空文字列""を指定して下さい。(デフォルト)
str_num(個数文字)は中心線と2点間以外の場合有効です。
upper_tolword 上段公差値文字
省略時は空文字列""を指定して下さい。(デフォルト)
val_pos 文字任意位置指定。
true: 任意位置ON（配置点位置に寸法値を配置します）
false: 任意位置OFF（寸法値を自動配置します）(デフォルト)
txt_pos（文字任意位置）は直径／両側、直径／片側、直径／２点間、直径／中心線、
半径／標準、半径／中心線のみ有効。
