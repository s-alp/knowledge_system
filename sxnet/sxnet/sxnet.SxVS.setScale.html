SxVS.setScale メソッド 

ビュー/ローカルビュー/子図の尺度を設定します。

[Visual Basic]
Public Sub setScale( _
   ByVal scale As Double, _
   ByVal move_view As Boolean _
)
[C#]
public void setScale(
   double scale,
   bool move_view
);

パラメータ

scale
尺度
move_view
ビュー間隔変更(ローカルビュー、子図の場合無効です)
true:ビューの配置間隔を変更します。
false:ビューの配置間隔を変更しません。

解説

ビュー/ローカルビュー/子図の尺度を変更します。
グローバルビューは処理対象外です。
自VSを表示するウィンドウが存在しない場合、新規にウィンドウを開設します。
自VSを表示するウィンドウがアクティブウィンドウに設定されます。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
