SxInfLayer メンバ

SxInfLayer 概要

パブリック コンストラクタ

SxInfLayer コンストラクタ 指定レイヤ情報のコピーを作成します。

パブリック フィールド

is_disp 表示属性。
true:表示レイヤ
false:非表示レイヤ
is_search 検索属性。
true:検索レイヤ
false:非検索レイヤ
name レイヤ名。未設定の場合は空文字列""です。
no レイヤ番号(1〜255)
