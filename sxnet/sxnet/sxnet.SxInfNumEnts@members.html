SxInfNumEnts メンバ

SxInfNumEnts 概要

パブリック コンストラクタ

SxInfNumEnts コンストラクタ 指定SxInfNumEntsインスタンスのコピーを作成します。

パブリック フィールド

layer レイヤ番号
layerが0以外の場合、nameは無効です。
n_ent 要素数
name 要素タイプ名
nameが空文字("")でない場合、layerの値は無効です。
