戻り値

セグメントのジオメトリ

解説

自要素のジオメトリを問い合わせます。
セグメントのタイプにより、以下のオブジェクトを返します。
type=SEGTYPE_POINTの場合、SxGeomPoint3D又はSxGeomPoint2D
type=SEGTYPE_LINEの場合、SxGeomLine3D又はSxGeomLine2D
type=SEGTYPE_CIRの場合、SxGeomCircle3D又はSxGeomCircle2D
type=SEGTYPE_ARCの場合、SxGeomArc3D又はSxGeomArc2D
type=SEGTYPE_SPLの場合、SxGeomSpline2D
type=SEGTYPE_ELPの場合、SxGeomEllipse2D
type=SEGTYPE_ELPAの場合、SxGeomElparc2D
type=SEGTYPE_DLINの場合、SxGeomLengthDim
type=SEGTYPE_DANGの場合、SxGeomAngDim
type=SEGTYPE_DARCの場合、SxGeomDiaDim
type=SEGTYPE_TEXTの場合、SxGeomText
type=SEGTYPE_LBLの場合、SxGeomLabel
type=SEGTYPE_DCHAの場合、SxGeomChamDim
type=SEGTYPE_SMRKの場合、SxGeomSmark
type=SEGTYPE_DELTAの場合、SxGeomDelta
type=SEGTYPE_BALLの場合、SxGeomBalloon
type=SEGTYPE_HATCHの場合、SxGeomHatch
type=SEGTYPE_HATCH_EXの場合、SxGeomHatch
type=SEGTYPE_WELDの場合、SxGeomWeld
type=SEGTYPE_GTOLの場合、SxGeomTol
type=SEGTYPE_APLの場合、SxGeomAplDim
type=SEGTYPE_SYMの場合、SxGeomCutLine 又はSxGeomArrowView又はSxGeomSymbol又はSxGeomSymbolMetal
type=SEGTYPE_LEADの場合、SxGeomArrow
type=SEGTYPE_ARLの場合、SxGeomArcLengDim
type=SEGTYPE_FMRKの場合、SxGeomFinishMark

文字列(type=SEGTYPE_TEXT) および 注記(type=SEGTYPE_LBL)にて 「iCAD SXで使用できる他言語文字のフォント」を使用している場合、 OSの言語と異なる文字(日本語・中国語・韓国語）を正しく返答します。
上記以外のタイプのセグメントの場合、null(VBの場合、Nothing)を返答します。
立体、ソリッド、FCソリッドは処理対象外です。
この場合は、 各フェイス、エッジ単位で、 SxFaceオブジェクトのgetGeomメソッド、又は、 SxEdgeオブジェクトのgetGeomメソッドで、フェイス、 エッジのジオメトリを問い合わせる事ができます。

2次元線分のジオメトリを取得する。[C#] 

/* 自要素が2次元線要素の場合、線分の方向ベクトルを返します。*/
SxVec GetLineVec(SxEntSeg seg)
{
    SxGeom geom = null;
    SxGeomLine2D geom_line2d = null;
    SxVec vec = null;

    // 要素のジオメトリを取得します。
    geom = seg.getGeom();
    // ジオメトリが2次元線分の場合、方向ベクトルを取り出します。
    if ( geom.type == SxGeom.TYPE_LINE2D ){
        geom_line2d = (SxGeomLine2D)geom;
        vec = new SxVec(geom_line2d.vec);
    }

    return vec; // 方向ベクトル、又は、nullを返します。
}
