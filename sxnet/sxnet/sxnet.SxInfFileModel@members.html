SxInfFileModel 概要

パブリック コンストラクタ

SxInfFileModel コンストラクタ 指定図面ファイル情報のコピーを作成します。

パブリック フィールド

comment 図面コメント
exc 排他状態
true:排他制御中です。(現在、編集中の図面ファイル)
false:排他制御中ではありません。(開かれていない、又は、読み取り専用で開かれています)
level 図面ファイル レベル
n_vs VS数
拡張子が.icd形式以外のファイルの場合0です。
n_wf WF数
拡張子が.icd形式以外のファイルの場合0です。
owner 排他制御を行っているユーザ名又はマシン名(最大8文字まで)
version 図面ファイル バージョン
