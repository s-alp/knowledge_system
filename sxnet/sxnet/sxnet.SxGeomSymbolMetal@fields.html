SxGeomSymbolMetal フィールド

SxGeomSymbolMetal クラスのフィールドは以下のとおりです。SxGeomSymbolMetal クラスの完全なメンバ一覧については、 SxGeomSymbolMetal メンバ のトピックを参照してください。

パブリック フィールド

color 図形の色
cs 平面の定義座標系情報
2次元要素の場合無効です。
pnt1 記号始端点座標
pnt2 記号終端点座標
pnt3 記号幅指示点座標
symbol_type シンボル記号タイプ。 SxGeomSymbol.SYMBOL_METAL : 基準金
SxGeomSymbol.SYMBOL_NAIL_ZIP : 爪チャック
type (SxGeom から継承されます) ジオメトリタイプ
width 図形の線幅
SxSys.WIDTH_BOLD : 太線
SxSys.WIDTH_MIDDLE : 中線
SxSys.WIDTH_THIN : 細線
