SX.Net Class Library

SxOptSelect.ent_mode フィールド

要素選択モード。
ent=trueの場合の要素選択モードです。

SxSys.ENTMODE_SEG:実像部品/パーツのメンバセグメントを選択した場合に、 セグメントを返答します。
SxSys.ENTMODE_PART:実像部品/パーツのメンバセグメントを選択した場合に、 実像部品/パーツを返答します。
SxSys.ENTMODE_MULTI(=2):実像部品/パーツのメンバセグメントを選択した場合に、 CAD上でのコーディネートメニューの検索モード[部]/[要]の状態に従って、 セグメント又は実像部品/パーツを返答します。
