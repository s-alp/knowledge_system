SxEnt.setTranslucent メソッド (SxEnt[], Boolean)

半透明表示に一括変更します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

パラメータ

entlist
要素の配列
mode
処理モード
true:半透明表示
false:通常表示

戻り値

変更後の要素の配列

解説

指定3D要素群の半透明表示を一括変更します。
entlistには、アクティブWF内のパーツ／立体セグメント／ソリッド／F.C.ソリッドを指定して下さい。
半透明要素は、非検索F.C.ソリッドになります。
mode = trueの場合、立体セグメント／ソリッドはF.C.ソリッドになりますから、entlist[]内のインスタンスは無効になります。
