SxEnt.getExtent メソッド 

要素のエクステント情報を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function getExtent() As SxBox
[C#]
public SxBox getExtent();

戻り値

エクステント情報

解説

自要素のエクステントボックスを問い合わせます。
エクステントは、自要素を包含する正確な最小限のボックスである事は保証しません。
自要素を包含するボックスの目安と考えて下さい。
非表示の要素、又は、エクステントを持たない要素の場合、null(VBの場合Nothing)を返答します。
3Dパーツの場合、子パーツも含むエクステントを返答します。
要素が、非表示レイヤ、非検索レイヤに属していても、エクステントを返します。
