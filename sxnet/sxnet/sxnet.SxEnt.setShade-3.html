SxEnt.setShade メソッド (Boolean, Int32)

要素(3次元セグメント/パーツ)のCG表示状態の変更(CG表示モード指定/WIRE表示)
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Sub setShade( _
   ByVal shade As Boolean, _

)
[C#]
public void setShade(
   bool shade,
   int mode
);

パラメータ

shade
表示方法
true : CG表示
false : WIRE表示
mode
CG表示の時の種別(shade=true時のみ有効です)
SxSys.CG_MODE_NONE : 現在設定されているCG表示種別を引き継ぎます。
SxSys.CG_MODE_SHADE : 縁なしでのシェーディング表示。
SxSys.CG_MODE_EDGE : 縁ありでのシェーディング表示。
SxSys.CG_MODE_HIDDEN : 隠線表示 。

解説

自要素の表示状態(CG表示/WIRE表示)の変更を行います。
自要素が3Dセグメント、3Dパーツ以外の場合、処理は行われません。
modeの指定は、shade=trueの場合のみ有効です。
システムのCG表示モードが指定したモードに切り替わるので、 他のCG表示中の要素の表示状態も同時に変更されます。
自要素を表示するウィンドウがアクティブに設定されます。
