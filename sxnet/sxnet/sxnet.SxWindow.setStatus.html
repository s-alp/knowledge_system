SxWindow.setStatus メソッド 

ウィンドウの表示状態を変更します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Sub setStatus( _
   ByVal status As Integer _
)
[C#]
public void setStatus(
   int status
);

パラメータ

status
表示状態。
SxWindow.STATUS_NORMAL：通常表示
SxWindow.STATUS_ICON:最小化(アイコン化)
SxWindow.STATUS_MAX:最大化

解説

自ウィンドウの表示状態の変更を行います。
既に他のウィンドウが最大表示中に、最大化を指定した場合、 他のウィンドウを通常表示に戻して、自ウィンドウを最大表示します。
最大化、通常表示の場合、自ウィンドウがアクティブウィンドウになります。
最小化を行った場合、他のウィンドウがアクティブウィンドウになります。
最大化されたウィンドウを最小化した後に、通常表示を行った場合、最大表示に戻ります。
