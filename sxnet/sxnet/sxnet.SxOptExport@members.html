パブリック コンストラクタ

SxOptExport コンストラクタ デフォルト設定でインスタンスを初期化します。

パブリック フィールド

file_type 図面変換ファイル種別。
SxOptExport.FILE_TYPE_DXF : DXFファイル
SxOptExport.FILE_TYPE_IGES : IGESファイル
SxOptExport.FILE_TYPE_BMI : BMIファイル
SxOptExport.FILE_TYPE_DWG : DWGファイル
SxOptExport.FILE_TYPE_DWF : DWFファイル
SxOptExport.FILE_TYPE_DOS : DOSファイル
SxOptExport.FILE_TYPE_STL : STLファイル(シングルボディ)
SxOptExport.FILE_TYPE_STL_MULTI : STLファイル(マルチボディ)
SxOptExport.FILE_TYPE_WRL : WRL(VRML)ファイル
SxOptExport.FILE_TYPE_VPS : FJVPSファイル
SxOptExport.FILE_TYPE_STEP : STEPファイル
SxOptExport.FILE_TYPE_PS : Parasolidファイル
初期値=0(無効な値)
ps_color Parasolidファイル出力時の色反映モード。
true : 反映する。(デフォルト)
false : 反映しない。
file_type=SxOptExport.FILE_TYPE_PS時のみ有効です。
ps_format Parasolidファイル拡張子タイプ。
SxOptExport.PS_FORMAT_X_T : x_t (デフォルト値)
SxOptExport.PS_FORMAT_XMT_TXT : xmt_txt
SxOptExport.PS_FORMAT_X_B : x_b
SxOptExport.PS_FORMAT_XMT_BIN : xmt_bin
file_type=SxOptExport.FILE_TYPE_PS時のみ有効です。
ps_refer Parasolidファイル出力時のパーツ部品参照モード。
true : 参照出力する。
false : 参照出力しない。(デフォルト)
file_type=SxOptExport.FILE_TYPE_PS、かつ、ps_transf=true時のみ有効です。
ps_transf Parasolidファイル出力時のパーツ配置情報モード。
true : 出力する。(デフォルト)
false : 出力しない。
file_type=SxOptExport.FILE_TYPE_PS時のみ有効です。
ps_version Parasolidファイルバージョン(8〜)
file_type=SxOptExport.FILE_TYPE_PS時のみ有効です。
初期値=13
stl_arc_div STL出力時の円弧出力精度。
0 : 指定なし(デフォルト)
0以外の場合は、以下の分割数を設定して下さい。
18,24,36,45,60,72,90,120,180,360
file_type=SxOptExport.FILE_TYPE_STL, SxOptExport.FILE_TYPE_STL_MULTI時のみ有効です。
stl_coord STL出力時の座標系。
true : STL座標系(デフォルト) :Z軸が天頂方向。
false : システム座標系 : CADと同一の座標系。(Y軸が天頂方向)
file_type=SxOptExport.FILE_TYPE_STL, SxOptExport.FILE_TYPE_STL_MULTI時のみ有効です。
