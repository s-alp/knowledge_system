SxInfEnt メンバ

SxInfEnt 概要

静的 パブリック (共有) フィールド

BODY_OTHER ボディタイプ定数：その他
BODY_SHEET ボディタイプ定数：シートボディ
BODY_SOLID ボディタイプ定数：ソリッドボディ(中身の詰まった立体)
BODY_WIRE ボディタイプ定数：ワイヤボディ
FC_STATE_ELEMENT FCソリッド状態定数：構成要素表現
FC_STATE_NONE FCソリッド状態定数：FCソリッド以外の要素
FC_STATE_NONMANIFOLD FCソリッド状態定数：非多様体
FC_STATE_NORMAL FCソリッド状態定数：部品形状表現
KIND_GROUP 要素種別定数：グループ
KIND_INVALID 要素種別定数：無効な要素
KIND_PART 要素種別定数：3次元パーツ
KIND_REFER 要素種別定数：レファー（配置子図、写像部品）
KIND_RPART 要素種別定数：実像部品
KIND_SEGMENT 要素種別定数：セグメント(一般セグメント、FCソリッド、ソリッド)
MEMBER_GRP 要素分類定数：グループ又は実像部品に属する
MEMBER_NONE 要素分類定数：何にも属さない

パブリック コンストラクタ

SxInfEnt コンストラクタ 指定要素情報のコピーを作成します。

パブリック フィールド

body_type 要素がセグメントの場合のボディタイプ
SxInfEnt.BODY_SOLID:ソリッドボディ
SxInfEnt.BODY_SHEET:シートボディ
SxInfEnt.BODY_WIRE:ワイヤボディ又は点
SxInfEnt.BODY_OTHER:その他
要素がセグメント以外の場合、BODY_OTHERです。
dim 要素の次元
2:2次元
3:3次元
fc_state FCソリッド状態
SxInfEnt.FC_STATE_NONE:FCソリッド以外の要素
SxInfEnt.FC_STATE_NORMAL:部品形状表現
SxInfEnt.FC_STATE_ELEMENT:構成要素表現
SxInfEnt.FC_STATE_NONMANIFOLD:非多様体
要素がF.C.ソリッド以外の場合、SxInfEnt.FC_STATE_NONEになります。
is_draft 下書き線属性
false:下書き線OFF
true:下書き線ON
is_transparent 半透明属性
false:通常表示
true:半透明表示
kind 要素種別
SxInfEnt.KIND_SEGMENT:セグメント(一般セグメント、FCソリッド、ソリッド)
SxInfEnt.KIND_RPART:実像部品
SxInfEnt.KIND_GROUP:グループ
SxInfEnt.KIND_REFER:レファー
SxInfEnt.KIND_PART:3Dパーツ
layer 要素の属するレイヤ番号。
グループ/実像部品/パーツの場合0になります。
member_kind 要素の分類
SxInfEnt.MEMBER_NONE:何にも属さない
SxInfEnt.MEMBER_GRP :グループ/実像部品のメンバ要素
model 属するモデル
part 要素の属する3次元パーツ。
パーツには属さない場合は、null(VBの場合、Nothing)です。
type 要素のタイプ番号
visi 表示属性
false:非表示 true:表示
グループ、パーツ、実像部品の場合、本パラメタは無効です。(常にfalseを返します)
vs 要素の属するVS
dim=3の場合は、null(VBの場合、Nothing)です。
wf 要素の属するWF
dim=2の場合は、null(VBの場合、Nothing)です。
