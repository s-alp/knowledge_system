解説

アクティブウィンドウに表示されたWF上に、 pnt_array[]で指定された基準面をzvec方向に垂直投影した投影体セグメントを作成します。
zvec,xvecは必ず直交している必要があります。
pnt_arrayは、定義座標系のXY平面座標系(2D座標)で表された、 投影体基準面の閉ループを指定して下さい。
1点目は、閉ループの開始座標であり、pnt_array[0].is_arc=falseに設定して下さい。
閉ループの自己交差は認められません。
又、始終点間以外の隣り合う座標間の距離は、0.0001以上離れている必要があります。
閉ループの座標数が64点を超える場合は、ソリッドで作成されます。
(円弧の場合は2点を消費します)

半円柱を作成する。[C#]

// 垂直投影体作成サンプル：半円柱を作成する。

SxCoord[] loop1 = new SxCoord[3]; // 基準面ループ情報
int n_pnt = loop1.Length;          // 構成座標数
double height = 20.0;              // 投影体高さ

// 基準面のループを定義する。
loop1[0] = new SxCoord( 0.0,  0.0, false,  0.0); // ループ始点
loop1[1] = new SxCoord(10.0,  0.0, true, 180.0); // 180度の円弧
loop1[2] = new SxCoord(loop1[0]);                 // ループ終点(=始点)

// 配置座標の指定(原点に配置)
SxPos org = new SxPos( 0.0, 0.0, 0.0 );

// 配置方向の指定(Y軸方向が高さ方向となるよう回転)
SxVec zvec = new SxVec( 0.0, 1.0, 0.0 );
SxVec xvec = new SxVec( 1.0, 0.0, 0.0 );

// 投影体作成
SxEntSeg seg = SxEntSeg.createSwept( org, zvec, xvec, height, loop1, n_pnt );
