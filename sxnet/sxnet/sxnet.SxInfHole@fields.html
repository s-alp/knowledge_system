SxInfHole フィールド

SxInfHole クラスのフィールドは以下のとおりです。SxInfHole クラスの完全なメンバ一覧については、 SxInfHole メンバ のトピックを参照してください。

パブリック フィールド

cs 穴の配置情報。
cs.orgが穴の配置位置、
cs.zvecが穴の深さ方向ベクトルです。
d_type 詳細タイプ
detail_name 穴の詳細名称
dir_th 巻方向
face_list 穴を構成するフェイスの配列。
FCソリッドが構成要素表現の場合、null(VBの場合、Nothing)です。
hole_number 穴番号
name 穴の名称
num_th 条数
param_list 穴の寸法情報の配列。
pattern_name 穴の呼び名称
penetrate 貫通属性
tap_type ねじ種類
u_name 予約パラメータ。
