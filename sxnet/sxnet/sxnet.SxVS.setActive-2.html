SxVS.setActive メソッド (Boolean)

アクティブVSに設定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Function setActive( _
   ByVal new_window As Boolean _
) As SxWindow
[C#]
public SxWindow setActive(
   bool new_window
);

パラメータ

new_window
新規ウィンドウ開設処理。
true:自VSを表示するウィンドウが存在しない場合、新規ウィンドウを開設します。
false:自VSを表示するウィンドウが存在しない場合、自モデルを表示するウィンドウの 表示を自VSに切り替えます。

戻り値

アクティブウィンドウ

解説

自VSをアクティブVS(アクティブビュー)に設定します。
自VSを表示するウィンドウがアクティブウィンドウに設定されます。
自VSを表示するウィンドウが存在しない場合、new_windowの指定に従って、 新規にウィンドウを開設するか、自モデルを表示するウィンドウの表示を自VSに 切り替えます。
ローカルビュー、基本ビューの場合、グローバルビューを表示するウィンドウを アクティブウィンドウに設定し、自VSをアクティブビューに設定します。

例外
