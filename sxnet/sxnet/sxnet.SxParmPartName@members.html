SxParmPartName メンバ

SxParmPartName 概要

パブリック コンストラクタ

SxParmPartName オーバーロード。 SxParmPartName クラスの新しいインスタンスを初期化します。

パブリック フィールド

comment 設定するコメント文字列。
半角英数字で最大80文字まで。(全角文字で最大40文字まで)
コメントを削除する場合は、空文字列("")を指定して下さい。
null(VBの場合Nothing)を指定した場合、コメントは変更しません。
filename 設定する参照ファイル名。(拡張子なし)
半角英数字で最大40文字まで。(全角文字で最大20文字まで)
外部パーツの場合のみ有効です。
空文字列("")で、パーツ名と同名の参照ファイル名を設定します。
null(VBの場合Nothing)を指定した場合、参照ファイル名は変更しません。
name 設定するパーツ名。
半角英数字で最大40文字まで。(全角文字で最大20文字まで)
null(VBの場合Nothing)を指定した場合、パーツ名は変更しません。
part 処理対象のパーツ
