SX.Net Class Library

SxVS.convPos メソッド 

座標変換(ビュー/ローカルビュー ←→ グローバルビュー)を行います。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function convPos( _
   ByVal mode As Boolean, _
   ByVal pos As SxPos _
) As SxPos
[C#]
public SxPos convPos(
   bool mode,
   SxPos pos
);

パラメータ

mode
変換モード。
true:自VS座標系→グローバルビュー座標系変換
false:グローバルビュー座標系→自VS座標系変換
pos
自VS又はグローバルビュー上の座標

戻り値

変換された座標

解説

指定座標の座標系の変換を行います。
自VSがビュー、ローカルビューでない場合、何も変換しません。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
