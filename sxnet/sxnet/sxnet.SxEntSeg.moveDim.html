寸法線の配置位置変更を行います。

[Visual Basic]
Public Sub moveDim( _
   ByVal pnt As SxPos, _
   ByVal mode As Integer, _
   ByVal term_mark As Boolean, _
   ByVal bolt As Boolean _
)
[C#]
public void moveDim(
   SxPos pnt,
   int mode,
   bool term_mark,
   bool bolt
);

パラメータ

pnt
配置点座標
mode
移動モード
SxEntSeg.MOVE_ANY : 任意位置に移動
SxEntSeg.MOVE_HOR : 横方向移動
SxEntSeg.MOVE_VER : 縦方向移動
溶接記号、表面粗さ、幾何公差の場合は、MOVE_ANYのみ指定可能です。
term_mark
端末記号追従
true:追従します。
false:追従しません。
寸法線以外の場合、本パラメタは無視されます。
bolt
稲妻半径寸法の稲妻終点変更
true:稲妻終点を変更します。
false:稲妻終点を変更しません。
稲妻半径寸法で、mode=SxEntSeg.MOVE_ANYの場合のみ有効です。
trueの場合は、稲妻終点座標を指定座標に変更します。
falseの場合は、寸法線の任意位置変更を行います。

解説

寸法線の配置位置の変更を行います。
自要素は、アクティブウィンドウに表示されたアクティブVS/WF上の要素である必要があります。
自要素が、以下のセグメントタイプの場合、処理対象となります。
注記(タイプ=SEGTYPE_LBL)
風船(タイプ=SEGTYPE_BALL)
長さ寸法線(タイプ=SEGTYPE_DLIN)
角度寸法線(タイプ=SEGTYPE_DANG
径寸法線(タイプ=SEGTYPE_DARC
面取寸法線(タイプ=SEGTYPE_DCHA
円弧長寸法(タイプ=SEGTYPE_ARL)
角寸/長円/角穴寸法線(タイプ=SEGTYPE_APL)
溶接記号(タイプ=SEGTYPE_WELD)
幾何公差(タイプ=SEGTYPE_GTOL)
表面粗さ(タイプ=SEGTYPE_SMRK)
