SxInfSearchEntType.reset メソッド 

全メンバフィールドの検索属性の設定を行います。

[Visual Basic]
Public Sub reset( _
   ByVal onoff As Boolean _
)
[C#]
public void reset(
   bool onoff
);

パラメータ

onoff
検索属性
true : 全メンバフィールドをtrue(検索)に設定します。
false : 全メンバフィールドをfalse(非検索)に設定します。

解説

draw,draft,solidの全メンバフィールトをonoffで指定された値に設定します。
自インスタンス内のフィールド値の設定を行うのみで、 CAD上の要素の検索タイプの状態は変わりません。
CAD上の検索要素タイプの設定を行うには、SxSysクラスの、 setSearchEntTypeメソッドを使用します。
