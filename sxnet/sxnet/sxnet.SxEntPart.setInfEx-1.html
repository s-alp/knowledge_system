SxEntPart.setInfEx メソッド (String)

自パーツに任意情報を設定します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Sub setInfEx( _
   ByVal info As String _
)
[C#]
public void setInfEx(
   string info
);

パラメータ

info
任意情報文字列(CSV形式)

解説

自パーツにのみCSV形式の任意情報文字列を設定します。
※全ての同名パーツに同じ任意情報文字列を設定してください。
infoは、半角英数字で最大4000文字までです。(全角文字で最大2000文字まで) 自パーツが既に任意情報を持つ場合、上書きされます。

文字列は、以下の形式で指定して下さい。
項目ID1,"内容1",項目ID2,"内容2", ...
例）User_type,"A",User_weight,"200.0"

本メソッドで付加した情報は、CAD本体の３次元ツール/部品表コマンドで参照されます。
CAD本体の部品表コマンドで利用する情報を付加する場合は、定義したテンプレート の内容に従って、文字列をCSV形式で指定して下さい。
部品表コマンドの詳細については、CAD本体のヘルプを参照して下さい。

[注意]
同名パーツに異なった任意情報文字列を設定した場合は、
３次元ツール/部品表コマンドや保存などが正しく動作しないことがあります。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外

使用例

[C#] 

void SetInf(SxEntPart part)
{
    // 付加する任意情報文字列の定義
    string info = "User_Volume,\"100.0\",User_Weight,\"200.0\"";
    // 任意情報を付加する。
    part.setInfEx(info);
}
[VB] 
Private Sub SetInf(ByVal part As SxEntPart)
    ' 付加する任意情報文字列の定義
    Dim info As String = "User_Volume,""100.0"",User_Weight,""200.0"""
    ' 任意情報を付加する。
    part.setInfEx(info)
End Sub
