解説

1994以前JIS規格表面粗さ記号を作成します。 objには、アクティブウィンドウに表示されたアクティブVS/WF上の要素を指定して下さい。
アクティブウィンドウが3次元で、かつ操作次元が2次元の場合は、アクティブな組立平面上の要素を 指定して下さい。
objには、以下のインスタンスが指定可能です。
SxPos(座標)
SxEdge(直線エッジ、円エッジ、円弧エッジ)
SxEntSeg(要素:線、円、楕円、円弧、楕円弧、スプライン)
objがSxPos場合、点配置の表面粗さ記号を作成します。pnt1（引出点座標）とpnt2（配置点座標）は無視されます。
objがSxEdgeあるいはSxEntSeg場合、要素配置の表面粗さ記号を作成します。
また、opt.lead_line = true かつ、opt.additionalline = falseの場合、pnt1（引出点座標）は無視されます。
2次元要素作成の場合、各座標のZ値は無視されます。
3次元上で組立平面が開設されていて、操作次元が2次元の場合、組立平面上に2次元要素を作成します。
操作次元が3次元の場合、メニューバー／３Ｄ設定／３Ｄ寸法線で設定した面上に3次元要素を作成します。
1994以前JIS規格表面粗さ記号作成オプションについては、SxOptSurfmarkクラスを参照して下さい。
