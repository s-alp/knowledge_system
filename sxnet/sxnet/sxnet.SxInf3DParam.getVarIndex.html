SxInf3DParam.getVarIndex メソッド 

指定パラメタ名を持つSxInf3DParamインスタンスの配列内インデックス番号を問い合わせます。

パラメータ

var_name
パラメタ名

検索を行うSxInf3DParamインスタンスの配列

戻り値

配列内のインデックス番号(>=0)
見つからなかった場合は、-1を返します。

解説

指定パラメタ名を持つSxInf3DParam配列内のインデックス番号を問い合わせます。
指定パラメタ名を持つインスタンスが見つからない場合は、-1を返します。

使用例

アクティブWFに定義された、パラメトリック情報配列を取得し、 パラメタ名"L1"の情報の、配列内インデックス番号を問い合わせる。
[C#] 

// アクティブWFに、パラメタ名"L1"を持つ、3Dパラメトリック条件が
// 定義されているか、問い合わせる。
// 未定義の場合、-1を返す。定義されている場合は、0以上の整数を返す。
int GetIndexL1()
{
    // アクティブWFの取得
    SxWF wf = SxWF.getActive()
    // アクティブウィンドウにWFを表示していなければ、終了。
    if ( wf == null )
        return -1; 

    // 3Dパラメトリック情報の取得
    SxInf3DParam[] list = wf.getInf3DParametric();
    // 3Dパラメトリック情報が定義されていなければ、終了。
    if ( list == null )
        return -1;

    // パラメタ名"L1"の、配列内INDEXを問い合わせる。
    // ※存在しない場合は、-1が返る。
    int index = SxInf3DParam.getVarIndex("L1", list);

    // 終了
    return index;
}
