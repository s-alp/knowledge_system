SxSys.getInfNumParts メソッド (Boolean)

アクティブウィンドウに表示されているモデル(図面)の部品数を問い合わせます。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Overloads Public Shared Function getInfNumParts( _
   ByVal allview As Boolean _
) As SxInfNumParts()
[C#]
public static SxInfNumParts[] getInfNumParts(
   bool allview
);

パラメータ

allview
処理モード:
true:全ビューを対象にする.
false:アクティブビューを対象にする.
※アクティブウィンドウが2Dの場合のみ有効です。

戻り値

部品数情報インスタンスの配列

解説

アクティブウィンドウが2Dの場合、実像部品、写像部品の集計を行います。
アクティブウンンドウが3Dの場合、パーツの集計を行います。
部品が存在しない場合、null(VBの場合、Nothing)を返します。
