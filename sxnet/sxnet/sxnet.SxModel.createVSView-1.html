基本ビューを開設します。

パラメータ

viewtype
基本ビュータイプ。
SxVS.VIEW_FRONT : 正面図
SxVS.VIEW_BACK : 背面図
SxVS.VIEW_TOP : 平面図
SxVS.VIEW_BOTTOM : 下面図
SxVS.VIEW_LEFT : 左側面図
SxVS.VIEW_RIGHT : 右側面図
distno
ビュー間隔番号 (1〜5)
CADでの基本ビューコマンド自動配置でのビュー間隔1〜5の指定に同じです。
既に、基準ビュー(XYビュー)以外のビューが配置されている場合、本パラメタは無視されます。

戻り値

開設したVS(基本ビュ−)インスタンス

解説

自モデル内に基本ビューを開設します。
自モデルがアクティブモデルとなります。
開設したVSがアクティブVSとなります。
自モデルの基本ウィンドウが3次元を表示している場合、2次元に次元変更されます。
指定ビューが既に開設されている場合、指定ビューをアクティブVSに設定します。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
