SxVS.getReferList メソッド 

自VS内のレファー(写像部品/配置子図)を取得します。
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

[Visual Basic]
Public Function getReferList( _
   ByVal visi As Boolean _
) As SxEntRefer()
[C#]
public SxEntRefer[] getReferList(
   bool visi
);

パラメータ

visi
非表示要素の処理。
true:返答する
false:返答しない

戻り値

SxEntReferインスタンスの配列
返答する要素が存在しない場合、null(VBの場合、Nothing)を返答します。

解説

自VS内に配置されている、写像部品、配置子図を取得します。
非検索レイヤの配置子図/写像部品は返答しません。
CAD上で配置子図/写像部品が非検索に設定されている場合、それぞれの要素は返答されません。

例外

例外の種類 条件
SxException CAD処理中の例外
Exception その他の例外
