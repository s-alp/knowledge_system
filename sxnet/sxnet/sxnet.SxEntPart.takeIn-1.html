SxEntPart.takeIn メソッド (Boolean, Boolean)

外部パーツの取り込み(内部パーツ化)を行います。（全階層指定）

[Visual Basic]
Overloads Public Sub takeIn( _
   ByVal all As Boolean, _
   ByVal alllevel As Boolean _
)
[C#]
public void takeIn(
   bool all,
   bool alllevel
);

パラメータ

all
同一外部パーツの一括取り込み
true:一括して取り込みます。
false:自パーツのみ取り込みます。
alllevel
全階層処理。
true:下位の外部パーツも処理します。
false:下位の外部パーツを処理しません。

解説

自外部パーツを内部パーツに変換します。
alllevel=trueの場合、自パーツ配下の下位の外部パーツも含めて処理します。
自パーツは外部パーツである必要があります。
自パーツが属するWFをアクティブに設定します。
入力パーツは、WFの最上位パーツに変更されます。
自パーツを表示するウィンドウが存在しない場合、新規にウィンドウを 開設します。
