SxEntSeg.createRadiusDimStd2D メソッド 

半径寸法線を作成します。(2次元)

[Visual Basic]
Public Shared Function createRadiusDimStd2D( _
   ByVal pos As SxPos, _
   ByVal arcinf As Double() _
) As SxEntSeg
[C#]
public static SxEntSeg createRadiusDimStd2D(
   SxPos pos,
   double[] arcinf
);

パラメータ

pos
寸法配置点座標
arcinf
寸法線の基準円／円弧情報
arcinf[0] :基準円/円弧の中心点のＸ座標
arcinf[1] :基準円/円弧の中心点のＹ座標
arcinf[2] :システム予約領域(0.0を指定して下さい)
arcinf[3] :基準円/円弧の半径
arcinf[4] :基準円/円弧の始角（単位:ラジアン。円の場合は0.0を指定して下さい）
arcinf[5] :基準円/円弧の相対角（単位:ラジアン。円の場合は2πを指定して下さい）

戻り値

作成した半径寸法線セグメント

解説

アクティブウィンドウに表示されたアクティブVSに、2次元半径寸法線を作成します。
posは、2次元の座標で、Z値は無視されます。
