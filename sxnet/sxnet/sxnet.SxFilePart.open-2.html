パラメータ

mode
処理モード
SxFilePart.MODE_SEG :実要素化
SxFilePart.MODE_PART:パーツ化
SxFilePart.MODE_GRP :グループ化
pos
配置座標
zvec
配置ベクトル(Z軸)
xvec
配置ベクトル(X軸)
layer
レイヤ引継ぎ(true:引き継ぐ, false:入力レイヤに作成)

戻り値

配置した部品
mode = SxFilePart.MODE_SEGの場合、nullを返します。(部品配置は実行されます)
mode = SxFilePart.MODE_PARTの場合、SxEntPartインスタンスを返します。
mode = SxFilePart.MODE_GRPの場合、SxEntGrpインスタンスを返します。

解説

3次元標準部品を、アクティブウィンドウに表示された、アクティブWFに配置します。
zvec,xvecはそれぞれ、配置時のZ軸、X軸に対する、傾きを表します。
zvec,xvecは直交している必要があります。
