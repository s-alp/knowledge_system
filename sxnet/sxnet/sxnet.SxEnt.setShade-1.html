SxEnt.setShade メソッド (SxEnt[], Boolean, Int32)

指定した3次元要素群のCG表示状態を変更します。(CG表示モード指定/WIRE表示)
このメソッドはiCAD SX ビューア／iCAD SX 組立支援の場合でも実行可能です。

パラメータ

entlist
表示状態を変更する3次元要素群の配列。
同一WF内の3Dセグメント、パーツを指定して下さい。
shade
表示方法
true : CG表示
false : WIRE表示
mode
CG表示の時の種別(shade=true時のみ有効です)
SxSys.CG_MODE_NONE : 現在設定されているCG表示種別を引き継ぎます。
SxSys.CG_MODE_SHADE : 縁なしでのシェーディング表示。
SxSys.CG_MODE_EDGE : 縁ありでのシェーディング表示。
SxSys.CG_MODE_HIDDEN : 隠線表示。

解説

指定要素群の表示状態(CG表示/WIRE表示)の変更を行います。
entlist[]には、同一WF内の要素を指定して下さい。又、グループは指定できません。
指定要素が3D立体形状でない場合は、表示状態は変化しません。
modeの指定は、shade=trueの場合のみ有効です。
システムのCG表示モードが指定したモードに切り替わるので、 他のCG表示中の要素の表示状態も同時に変更されます。
指定要素を表示するウィンドウがアクティブに設定されます。

アクティブウィンドウ内の全要素の表示状態を変更したい場合は、SxSysクラスの setShadeを使用して下さい。
