解説

アクティブウィンドウに表示されたWF上に、 キャップを作成します。
-aixis1/2 <= position <= axis1/2 である必要があります。
縦軸はZ軸方向です。
横軸はX軸方向です。
断面法線は、-Z軸方向です。
diameter=axis2で、楕円中心位置に断面を作成します。
reverse=trueかつ、diameter < axis2の場合、断面位置を楕円中心よりも下の位置で断面を作成します。
reverse=falseかつ、diameter < axis2の場合、断面位置を楕円中心よりも上の位置で断面を作成します。
